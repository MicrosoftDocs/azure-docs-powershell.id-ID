# Get-AzDiskAccess

## SYNOPSIS
Mendapatkan properti Akses Disk

## SINTAKS

### DefaultParameterSet (Default)
```
Get-AzDiskAccess [[-ResourceGroupName] <String>] [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIDParameterSet
```
Get-AzDiskAccess [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Get-AzDiskAccess** mendapatkan properti Disk Accesses

## CONTOH

### Contoh 1: Menggunakan Kumpulan Parameter Default 
```
PS C:\> Get-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -Name 'DiskAccess01'

Name                       : DiskAccess01
Type                       : Microsoft.Compute/diskAccesses
Tags                       : {}
```

Perintah ini mendapatkan properti sumber daya Akses Disk bernama 'DiskAccess01' dalam grup sumber daya 'ResourceGroup01'.
