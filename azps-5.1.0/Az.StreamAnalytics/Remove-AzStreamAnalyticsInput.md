# Remove-AzStreamAnalyticsInput

## SYNOPSIS
Menghapus input dari pekerjaan Analitik Stream.

## SINTAKS

```
Remove-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Remove-AzStreamAnalyticsInput** secara asinkron menghapus input dari pekerjaan Analitik Stream di Azure.

## CONTOH

### CONTOH 1: Menghapus aliran input dari pekerjaan
```
PS C:\>Remove-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

Perintah ini menghapus input EventStream dari StreamingJob.
