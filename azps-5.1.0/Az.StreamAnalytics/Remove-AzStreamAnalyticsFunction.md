# Remove-AzStreamAnalyticsFunction

## SYNOPSIS
Menghapus fungsi dari pekerjaan Stream Analytics.

## SINTAKS

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Remove-AzStreamAnalyticsFunction** menghapus fungsi secara asinkron dari pekerjaan Analitik Azure Stream.

## CONTOH

### Contoh 1: Menghapus fungsi Stream Analytics
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Perintah ini menghapus fungsi yang bernama ScoreTweet dari pekerjaan bernama StreamJob22.
