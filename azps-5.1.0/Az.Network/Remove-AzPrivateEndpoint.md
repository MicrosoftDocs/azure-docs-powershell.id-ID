# Remove-AzPrivateEndpoint

## SYNOPSIS
Menghapus titik akhir privat.

## SINTAKS

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Remove-AzPrivateEndpoint** menghapus titik akhir privat. 

## CONTOH

### Contoh 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

Contoh ini menghapus titik akhir privat bernama MyPrivateEndpoint1.
