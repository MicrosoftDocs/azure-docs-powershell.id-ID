## DESKRIPSI
Cmdlet **Edit-AzWebAppBackupConfiguration** mengedit cadangan konfigurasi saat ini untuk Azure Web App.

## CONTOH

### Contoh 1

Cmdlet Edit-AzWebAppBackupConfiguration mengedit cadangan konfigurasi saat ini untuk Azure Web App. (otomatisgenerated)

```powershell <!-- Aladdin Generated Example -->

```
