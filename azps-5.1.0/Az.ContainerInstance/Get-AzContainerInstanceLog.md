# Get-AzContainerInstanceLog

## SYNOPSIS
Mendapatkan log contoh wadah dalam grup wadah.

## SINTAKS

### GetContainerInstanceLogByNamesParamSet (Default)
```
Get-AzContainerInstanceLog [-ResourceGroupName] <String> -ContainerGroupName <String> [-Name <String>]
 [-Tail <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetContainerInstanceLogByPSContainerGroupParamSet
```
Get-AzContainerInstanceLog -InputContainerGroup <PSContainerGroup> [-Name <String>] [-Tail <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetContainerInstanceLogByResourceIdParamSet
```
Get-AzContainerInstanceLog -ResourceId <String> [-Name <String>] [-Tail <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Get-AzContainerInstanceLog** mendapatkan log wadah dalam grup wadah.

## CONTOH

### Contoh 1: Mendapatkan log arah contoh wadah
```
PS C:\> Get-AzContainerInstanceLog -ResourceGroupName demo -ContainerGroupName mycontainer -Name container1

Log line 1.
Log line 2.
Log line 3.
Log line 4.
```

Dapatkan log dari `container1` grup wadah `mycontainer` . Secara default, akan mengembalikan hingga 4MB konten log.

### Contoh 2: Mendapatkan log arah contoh wadah yang memiliki nama yang sama dengan grup wadah
```
PS C:\> Get-AzContainerInstanceLog -ResourceGroupName demo -ContainerGroupName mycontainer

Log line 1.
Log line 2.
Log line 3.
Log line 4.
```

Dapatkan log dari `mycontainer` grup wadah `mycontainer` . Secara default, akan mengembalikan hingga 4MB konten log.

### Contoh 3: Mendapatkan arah 2 baris log dari contoh wadah
```
PS C:\> Get-AzContainerInstanceLog -ResourceGroupName demo -ContainerGroupName mycontainer -Name container1 -Tail 2

Log line 3.
Log line 4.
```

Dapatkan arah 2 baris log dari `container1` dalam grup `mycontainer` wadah.

### Contoh 4: Mendapatkan log arah contoh wadah dalam grup pipa dalam wadah
```
PS C:\> Get-AzContainerGroup -ResourceGroupName demo -Name mycontainer | Get-AzContainerInstanceLog

Log line 1.
Log line 2.
Log line 3.
Log line 4.
```

Dapatkan log dari `mycontainer` dalam grup wadah di `mycontainer` pipa. Secara default, akan mengembalikan hingga 4MB konten log.
