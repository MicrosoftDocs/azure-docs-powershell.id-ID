# New-AzCosmosDBSqlUniqueKey

## SYNOPSIS
Membuat objek CosmosDB Sql UniqueKey baru.

## SINTAKS

```
New-AzCosmosDBSqlUniqueKey -Path <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **New-AzCosmosDBSqlUniqueKey** membuat objek baru tipe PSUniqueKey.

## CONTOH

### Contoh 1
```powershell
PS C:\> New-AzCosmosDBSqlUniqueKey -Path {path}

Path
----
{path}
```
