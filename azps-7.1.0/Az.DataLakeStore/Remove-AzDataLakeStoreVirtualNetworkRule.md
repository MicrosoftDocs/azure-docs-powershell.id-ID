# Remove-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Menghapus aturan jaringan virtual tertentu di Data Lake Store yang ditentukan.

## SYNTAX

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzDataLakeStoreVirtualNetworkRule** menghapus aturan jaringan virtual tertentu di Penyimpanan Data Lake yang ditentukan.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

Menghapus aturan jaringan virtual "myVNET" dari akun "dls"
