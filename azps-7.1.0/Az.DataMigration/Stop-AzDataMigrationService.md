# Stop-AzDataMigrationService

## SYNOPSIS
Menghentikan contoh Layanan Migrasi Database Azure yang berada dalam status berjalan.

## SYNTAX

### ComponentNameParameterSet (Default)
```
Stop-AzDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Stop-AzDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Stop-AzDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Stop-AzDataMigrationService menghentikan instans Layanan Migrasi Database Azure yang dalam status berjalan.

## EXAMPLES

### Contoh 1
```
PS C:\> Stop-AzDataMigrationService -ResourceGroupName MyResourceGroup -ServiceName TestService
```

Contoh di atas menghentikan contoh Layanan Migrasi Database Azure yang disebut TestService berdasarkan nama layanan yang diberikan sebagai parameter input

### Contoh 2
```
PS C:\> Stop-AzDataMigrationService -InputObject $TestService
```

Contoh di atas menghentikan contoh Layanan Migrasi Database Azure berdasarkan objek PSDataMigrationService yang diteruskan sebagai parameter input.
