# Remove-AzConnectedKubernetes

## SYNOPSIS
Hapus kluster tersambung, hapus sumber daya terlacak di Azure Resource Manager (ARM).

## SYNTAX

### Hapus (Default)
```
Remove-AzConnectedKubernetes -ClusterName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-KubeConfig <String>] [-KubeContext <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzConnectedKubernetes -InputObject <IConnectedKubernetesIdentity> [-KubeConfig <String>]
 [-KubeContext <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Hapus kluster tersambung, hapus sumber daya terlacak di Azure Resource Manager (ARM).

## EXAMPLES

### Contoh 1: Menghapus file tersambung
```powershell
PS C:\> Remove-AzConnectedKubernetes -ClusterName azps_test_cluster -ResourceGroupName azps_test_group

```

Perintah ini menghapus hubungan yang tersambung

### Contoh 2: Menghapus objek terhubung dengan objek
```powershell
PS C:\> Get-AzConnectedKubernetes -ClusterName azps_test_cluster -ResourceGroupName azps_test_group | Remove-AzConnectedKubernetes

```

Perintah ini menghapus objek tersambung ke objek
