# Remove-AzMySqlFlexibleServer

## SYNOPSIS
Menghapus server.

## SYNTAX

### Hapus (Default)
```
Remove-AzMySqlFlexibleServer -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzMySqlFlexibleServer -InputObject <IMySqlIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Menghapus server.

## EXAMPLES

### Contoh 1: Hapus server MySql berdasarkan resourceGroup dan nama server
```powershell
PS C:\> Remove-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -Name mysql-test

```

Cmdlet ini menghapus server MySql berdasarkan resourceGroup dan nama server.

### Contoh 2: Hapus server MySql berdasarkan identitas
```powershell
PS C:\> $ID = "/subscriptions/<SubscriptionId>/resourceGroups/PowershellMySqlTest/providers/Microsoft.DBforMySQL/flexibleServers/mysql-test"
PS C:\> Remove-AzMySqlFlexibleServer -InputObject $ID
 
```

Cmdlets ini menghapus server MySql berdasarkan identitas.
