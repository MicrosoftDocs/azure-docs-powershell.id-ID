# Unregister-AzAutomationDscNode

## SYNOPSIS
Menghapus simpul DSC dari manajemen dengan akun Otomatisasi.

## SYNTAX

```
Unregister-AzAutomationDscNode -Id <Guid> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Unregister-AzAutomationDscNode** menghapus simpul Konfigurasi Status Yang Diinginkan (DSC, Desired State Configuration) APS dari manajemen oleh akun Otomatisasi Azure.

## EXAMPLES

### Contoh 1: Hapus node Azure DSC dari manajemen dengan akun Otomatisasi
```
PS C:\>Unregister-AzAutomationDscNode -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111ca86067j8
```

Perintah ini menghapus node DSC yang memiliki GUID tertentu dari manajemen oleh akun Otomatisasi yang bernama Contoso17.
