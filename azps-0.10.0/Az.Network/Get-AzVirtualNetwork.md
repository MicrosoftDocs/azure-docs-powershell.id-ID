# Get-AzVirtualNetwork

## SYNOPSIS
Mendapatkan jaringan virtual dalam grup sumber daya.

## SYNTAX

### NoExpand
```
Get-AzVirtualNetwork [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Memperluas
```
Get-AzVirtualNetwork -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzVirtualNetwork** mendapatkan satu atau beberapa jaringan virtual n grup sumber daya.

## EXAMPLES

### 1: Mengambil jaringan virtual
```
Get-AzVirtualNetwork -Name MyVirtualNetwork -ResourceGroupName TestResourceGroup
```

Perintah ini mendapatkan jaringan virtual bernama MyVirtualNetwork dalam grup sumber daya TestResourceGroup
