# Get-AzExpressRouteCircuit

## SYNOPSIS
Mendapatkan sirkuit Azure ExpressRoute dari Azure.

## SYNTAX

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzExpressRouteCircuit** digunakan untuk mengambil objek sirkuit ExpressRoute dari langganan Anda. Objek sirkuit yang dikembalikan dapat digunakan sebagai input ke cmdlet lain yang beroperasi di sirkuit ExpressRoute.

## EXAMPLES

### Contoh 1: Dapatkan sirkuit ExpressRoute untuk dihapus
```
Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzExpressRouteCircuit
```
