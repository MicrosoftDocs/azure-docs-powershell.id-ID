# Remove-AzLoadBalancer

## SYNOPSIS
Menghapus penyeimbang muat.

## SYNTAX

```
Remove-AzLoadBalancer -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzLoadBalancer** menghapus penyeimbang muat Azure.

## EXAMPLES

### Contoh 1: Menghapus penyeimbang muat
```
PS C:\>Remove-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
```

Perintah ini menghapus penyeimbang muat yang bernama MyLoadBalancer dalam grup sumber daya bernama MyResourceGroup.
