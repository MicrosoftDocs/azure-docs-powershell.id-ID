# New-AzADUser

## SYNOPSIS
Membuat pengguna direktori aktif baru.

## SYNTAX

```
New-AzADUser -DisplayName <String> -UserPrincipalName <String> -Password <SecureString>
 [-ImmutableId <String>] [-MailNickname <String>] [-ForceChangePasswordNextLogin]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Membuat pengguna direktori aktif baru (akun kerja/sekolah juga dikenal sebagai org-id).
Untuk informasi selengkapnya: https://msdn.microsoft.com/en-us/library/azure/ad/graph/api/users-operations#CreateUser

## EXAMPLES

### Contoh 1 - Membuat pengguna AD baru
```
PS C:\> $SecureStringPassword = ConvertTo-SecureString -String "password" -AsPlainText -Force
PS C:\> New-AzADUser -DisplayName "MyDisplayName" -UserPrincipalName "myemail@domain.com" -Password $SecureStringPassword -MailNickname "MyMailNickName"
```

Membuat pengguna AD baru dengan nama "MyDisplayName" dan nama utama pengguna "myemail@domain.com" dalam penyewa.
