# New-AzADGroup

## SYNOPSIS
Membuat grup direktori aktif baru.

## SYNTAX

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Membuat grup direktori aktif baru.

## EXAMPLES

### Contoh 1 - Membuat grup AD baru

```
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Membuat grup AD baru dengan nama "MyGroupDisplayName" dan nama panggilan email "myemail@domain.com".
