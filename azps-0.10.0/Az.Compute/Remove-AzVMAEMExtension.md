# Remove-AzVMAEMExtension

## SYNOPSIS
Menghapus ekstensi AEM dari komputer virtual.

## SYNTAX

```
Remove-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [[-OSType] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzVMAEMExtension** menghapus ekstensi Azure Enhanced Monitoring (AEM) dari mesin virtual.

## EXAMPLES

### Contoh 1: Hapus ekstensi AEM
```
PS C:\> Remove-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Perintah ini akan menghapus ekstensi AEM untuk komputer virtual bernama contoso-server.
