# Remove-AzImageDataDisk

## SYNOPSIS
Menghapus disk data dari objek gambar.

## SYNTAX

```
Remove-AzImageDataDisk [-Image] <PSImage> [-Lun] <Int32> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzImageDataDisk** menghapus disk data dari objek gambar.

## EXAMPLES

### Contoh 1
```
PS C:\> Get-AzImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' | Remove-AzImageDataDisk -Lun 1 | Update-AzImage;
```

Perintah ini menghapus disk data unit logika nomor 1 dari gambar 'Gambar01' yang sudah ada dalam grup sumber daya 'ResourceGroup01'.
