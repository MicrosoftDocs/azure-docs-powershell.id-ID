# Remove-AzContainerService

## SYNOPSIS
Menghapus layanan wadah.

## SYNTAX

```
Remove-AzContainerService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzContainerService** menghapus layanan penampung dari akun Azure Anda.

## EXAMPLES

### Contoh 1: Menghapus layanan kontainer
```
PS C:\> Remove-AzContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17"
```

Perintah ini menghapus layanan wadah bernama CSResourceGroup17.
