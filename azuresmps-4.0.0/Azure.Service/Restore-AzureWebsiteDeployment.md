# Restore-AzureWebsiteDeployment

## SYNOPSIS
Menebus penyebaran situs web sebelumnya di Azure.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Restore-AzureWebsiteDeployment [-CommitId <String>] [-Force] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Topik ini menjelaskan cmdlet dalam versi 0.8.10 modul Microsoft Azure PowerShell.
Untuk mendapatkan versi modul yang Anda gunakan, di konsol Azure PowerShell, ketik .`(Get-Module -Name Azure).Version`

Cmdlet **Restore-AzureWebsiteDeployment** menampung ulang penyebaran situs web sebelumnya di Azure.
Proses ini menggantikan penyebaran saat ini dengan penyebaran yang dipilih.

## EXAMPLES

### Contoh 1: Redeploy situs
```
PS C:\> Restore-AzureWebsiteDeployment -Name "ContosoSite" -CommitId "f876543210"
```

Perintah ini menampung ulang penyebaran yang memiliki ID f876543210 untuk situs web bernama ContosoSite.
