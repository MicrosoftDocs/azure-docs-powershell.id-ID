# Get-AzureVMSqlServerExtension

## SYNOPSIS
Dapatkan pengaturan pengaturan SQL Server Agen IaaS di komputer virtual tertentu.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Get-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureVMSqlServerExtension** mendapatkan pengaturan infrastruktur SQL Server sebagai Agen layanan (IaaS) di komputer virtual tertentu.

## EXAMPLES

### Contoh 1: Dapatkan pengaturan ekstensi SQL Server di mesin virtual
```
PS C:\> Get-AzureVMSqlServerExtension-VM $VM
          ExtensionName        : SqlIaaSAgent
          Publisher            : Microsoft.SqlServer.Management
          Version              : 1.0
          State                : Enable
          RoleName             : VMName
          AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
          AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Dapatkan pengaturan ekstensi SQL Server di komputer virtual tertentu.

### Contoh 2: Mendapatkan pengaturan pengaturan SQL Server Agen IaaS di mesin virtual
```
PS C:\> Get-AzureVM -ServiceName "Service" -Name "VMName" | Get-AzureVMSqlServerExtension
          ExtensionName        : SqlIaaSAgent
          Publisher            : Microsoft.SqlServer.Management
          Version              : 1.0
          State                : Enable
          RoleName             : VMName
          AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
          AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Dapatkan pengaturan agen SQL Server IaaS di mesin virtual tertentu menggunakan input pipet.

### Contoh 3: Mendapatkan pengaturan perangkat SQL Server agen IaaS versi di komputer virtual
```
PS C:\> Get-AzureVMSqlServerExtension -VM $VM -Version "1.0"
          ExtensionName        : SqlIaaSAgent
          Publisher            : Microsoft.SqlServer.Management
          Version              : 1.0
          State                : Enable
          RoleName             : VMName
          AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
          AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Perintah ini mendapatkan pengaturan versi uji SQL Server IaaS Agen di komputer virtual.
