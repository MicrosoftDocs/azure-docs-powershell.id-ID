# New-AzureSiteRecoveryVault

## SYNOPSIS
Membuat vault layanan Pemulihan Situs.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
New-AzureSiteRecoveryVault -Name <String> -Location <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzureSiteRecoveryVault** membuat vault layanan Pemulihan Situs Azure.

## EXAMPLES

### Contoh 1: Membuat vault
```
PS C:\> New-AzureSiteRecoveryVault -Location "West US" -Name "ContosoVault" 
Response
--------
Vault has been created
```

Perintah ini membuat vault bernama ContosoVault di lokasi AS Barat.
