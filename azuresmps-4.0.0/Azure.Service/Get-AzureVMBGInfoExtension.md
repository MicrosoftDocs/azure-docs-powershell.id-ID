# Get-AzureVMBGInfoExtension

## SYNOPSIS
Dapatkan ekstensi BGInfo yang diterapkan pada mesin virtual.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Get-AzureVMBGInfoExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureVMBGInfoExtension** akan menerapkan ekstensi BGInfo pada mesin virtual.

## EXAMPLES

### Contoh 1: Dapatkan ekstensi BGInfo yang diterapkan pada mesin virtual tertentu
```
PS C:\> Get-AzureVMBGInfoExtension -VM $VM;
```

Perintah ini akan menerapkan ekstensi BGInfo pada mesin virtual tertentu
