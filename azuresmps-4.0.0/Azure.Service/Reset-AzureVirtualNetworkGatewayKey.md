# Reset-AzureVirtualNetworkGatewayKey

## SYNOPSIS
Mereset kunci gateway jaringan virtual.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Reset-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> -keyLength <Int32>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Reset-AzureVirtualNetworkGatewayKey** mengatur ulang kunci gateway jaringan virtual.
