# Add-AzureDns

## SYNOPSIS
Menambahkan server DNS ke layanan Azure.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Add-AzureDns [-Name] <String> [-IPAddress] <String> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Add-AzureDn** menambahkan server DNS ke layanan Azure.

## EXAMPLES

### Contoh 1: Menambahkan server DNS
```
PS C:\> Add-AzureDns -ServiceName "ContosoService" -IPAddress 10.1.2.4 -Name "Dns07"
```

Perintah ini menambahkan server DNS yang memiliki alamat 10.1.2.4 ke ContosoService.
