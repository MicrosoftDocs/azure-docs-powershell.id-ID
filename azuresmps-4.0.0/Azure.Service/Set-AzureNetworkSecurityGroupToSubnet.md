# Set-AzureNetworkSecurityGroupToSubnet

## SYNOPSIS
Mengaitkan grup keamanan jaringan ke subnet.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Set-AzureNetworkSecurityGroupToSubnet -Name <String> -VirtualNetworkName <String> -SubnetName <String> [-Force]
 [-PassThru] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzureNetworkSecurityGroupToSubnet** mengaitkan grup keamanan jaringan Azure ke subnet.
