# Get-AzureAutomationRunbook

## SYNOPSIS

Mendapatkan buku runbook.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

### SecaraSemua (Default)
```
Get-AzureAutomationRunbook -AutomationAccountName <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### ByRunbookName
```
Get-AzureAutomationRunbook -Name <String> -AutomationAccountName <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## DESCRIPTION

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

Cmdlet **Get-AzureAutomationRunbook** mendapatkan satu atau beberapa fitur Microsoft Azure runbooks Otomatisasi.
Secara default, semua runbooks akan dikembalikan.
Untuk mendapatkan runbook spesifik, tentukan nama atau ID-nya.
Untuk menautkan semua buku kerja ke jadwal tertentu, tentukan nama jadwal.

## EXAMPLES

### Contoh 1: Dapatkan semua runbooks
```
PS C:\> Get-AzureAutomationRunbook -AutomationAccountName "Contoso17"
```

Perintah ini mendapatkan semua runbooks dalam akun Azure Automation yang bernama Contoso17.
