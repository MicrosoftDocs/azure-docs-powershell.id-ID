# Get-AzureVMChefExtension

## SYNOPSIS
Mendapatkan ekstensi Chef diterapkan pada mesin virtual.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Get-AzureVMChefExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureVMChefExtension** mendapatkan ekstensi Chef yang diterapkan pada mesin virtual.

## EXAMPLES

### Contoh 1: Dapatkan ekstensi Chef yang diterapkan pada mesin virtual tertentu
```
PS C:\> Get-AzureVMChefExtension -VM $VM;
```

Perintah ini akan menerapkan ekstensi Chef pada mesin virtual tertentu.
