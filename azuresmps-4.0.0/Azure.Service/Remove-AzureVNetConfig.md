# Remove-AzureVNetConfig

## SYNOPSIS
Menghapus konfigurasi jaringan dari langganan Azure saat ini.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Remove-AzureVNetConfig [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureVNetConfig** menghapus semua pengaturan jaringan virtual dari langganan Azure saat ini.

## EXAMPLES

### Contoh 1: Menghapus konfigurasi jaringan virtual dari langganan saat ini
```
PS C:\> Remove-AzureVNetConfig
```

Perintah ini menghapus konfigurasi jaringan virtual dari langganan saat ini.
