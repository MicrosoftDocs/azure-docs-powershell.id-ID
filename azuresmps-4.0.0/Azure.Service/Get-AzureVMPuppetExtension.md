# Get-AzureVMPuppetExtension

## SYNOPSIS
Dapatkan ekstensi Samakan diterapkan pada mesin virtual.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Get-AzureVMPuppetExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureVMPuppetExtension** menerapkan ekstensi Pemutakhiran Pada mesin virtual.

## EXAMPLES

### Contoh 1: Dapatkan ekstensi Satu-satunya yang diterapkan pada mesin virtual
```
PS C:\> Get-AzureVMPuppetExtension -VM $VM
```

Perintah ini akan menerapkan ekstensi Samaso pada mesin virtual.
