# Remove-AzureRmEnvironment

## SYNOPSIS
Menghapus titik akhir dan metadata untuk terhubung ke instans Azure tertentu.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Remove-AzureRmEnvironment [-Name] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzureRmEnvironment menghapus titik akhir dan informasi metadata untuk terhubung ke contoh Azure tertentu.
