# Set-AzureStorageContainerStoredAccessPolicy

## SYNOPSIS
Menetapkan kebijakan akses tersimpan untuk tempat penyimpanan Azure.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Set-AzureStorageContainerStoredAccessPolicy [-Container] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-NoStartTime] [-NoExpiryTime] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>] [-ConcurrentTaskCount <Int32>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzureStorageContainerStoredAccessPolicy** mengatur kebijakan akses tersimpan untuk tempat penyimpanan Azure.

## EXAMPLES

### Contoh 1: Menetapkan kebijakan akses tersimpan dalam wadah penyimpanan
```
PS C:\>Set-AzureStorageContainerStoredAccessPolicy -Container "MyContainer" -Policy "Policy06"
```

Perintah ini mengatur kebijakan akses yang bernama Policy06 untuk wadah penyimpanan bernama MyContainer.
