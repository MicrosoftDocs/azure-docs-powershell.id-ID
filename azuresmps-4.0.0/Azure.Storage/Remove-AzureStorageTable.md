# Remove-AzureStorageTable

## SYNOPSIS
Menghapus tabel penyimpanan.

[!INCLUDE [rdfe-banner](../../includes/rdfe-banner.md)]

## SYNTAX

```
Remove-AzureStorageTable [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureStorageTable** menghapus satu atau beberapa tabel penyimpanan dari akun penyimpanan di Azure.

## EXAMPLES

### Contoh 1: Menghapus tabel
```
PS C:\>Remove-AzureStorageTable -Name "TableABC"
```

Perintah ini menghapus tabel.

### Contoh 2: Menghapus beberapa tabel
```
PS C:\>Get-AzureStorageTable table* | Remove-AzureStorageTable
```

Contoh ini menggunakan karakter wildcard dengan parameter *Nama* untuk mendapatkan semua tabel yang cocok dengan tabel pola lalu meneruskan hasilnya pada garis pipa untuk menghapus tabel.
