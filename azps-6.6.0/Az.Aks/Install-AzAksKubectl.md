# Install-AzAksKubectl

## SYNOPSIS
Unduh dan instal kubectl, alat baris perintah Kubernetes.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.aks/install-azakskubectl) untuk informasi terbaru.

## SYNTAX

```
Install-AzAksKubectl [-Destination <String>] [-Version <String>] [-DownloadFromMirror] [-PassThru] [-AsJob]
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [-SubscriptionId <String>]
 [<CommonParameters>]
```

## DESCRIPTION
Unduh dan instal kubectl, alat baris perintah Kubernetes.

## EXAMPLES

### Unduh dan instal versi terbaru kubectl
```powershell
PS C:\> Install-AzAksKubectl -Version latest
```
