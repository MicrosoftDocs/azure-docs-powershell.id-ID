# Remove-AzStorageSyncService

## SYNOPSIS
Perintah ini akan menghapus layanan sinkronisasi penyimpanan tertentu.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.storagesync/remove-azstoragesyncservice) untuk informasi terkini.

## SYNTAX

### StringParameterSet (Default)
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Perintah ini akan menghapus layanan sinkronisasi penyimpanan tertentu. Layanan sinkronisasi penyimpanan hanya dapat dihapus jika semua grup sinkronisasi dan server terdaftar yang terkandung dihapus terlebih dahulu.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

Perintah ini akan menghapus layanan sinkronisasi penyimpanan.
