# Remove-AzStorageSyncCloudEndpoint

## SYNOPSIS
Perintah ini akan menghapus titik akhir awan tertentu dari grup sinkronisasi. Tanpa setidaknya satu titik akhir awan, tidak ada titik akhir server lain dalam grup sinkronisasi ini yang dapat disinkronkan.

## SYNTAX

### StringParameterSet (Default)
```
Remove-AzStorageSyncCloudEndpoint [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncCloudEndpoint [-InputObject] <PSCloudEndpoint> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncCloudEndpoint [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Perintah ini akan menghapus titik akhir awan tertentu dari grup sinkronisasi. File Azure yang berbagi referensi titik akhir awan tidak tersentuh oleh proses ini. Perintah ini hanya ditujukan untuk pencodaan. Menghapus titik akhir awan adalah operasi perusak. Titik akhir server tidak dapat disinkronkan tanpa ada setidaknya satu titik akhir awan. Operasi ini tidak boleh dilakukan untuk mengatasi masalah sinkronisasi. Jika berbagi file Azure ini ditambahkan lagi ke grup sinkronisasi yang sama, sebagai titik akhir awan, masalah ini dapat menyebabkan file yang berkonflik dan konsekuensi lain yang tidak diinginkan.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Remove-AzStorageSyncCloudEndpoint -Force -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName" -Name "myCloudEndpointName"
```

Perintah ini akan menghapus titik akhir awan.
