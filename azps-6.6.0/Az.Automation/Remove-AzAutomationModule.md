# Remove-AzAutomationModule

## SYNOPSIS
Menghapus modul dari Otomatisasi.

## SYNTAX

```
Remove-AzAutomationModule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzAutomationModule** menghapus modul dari akun Otomatisasi dalam Azure Automation.

## EXAMPLES

### Contoh 1: Menghapus modul
```
PS C:\>Remove-AzAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule" -ResourceGroupName "ResourceGroup01"
```

Perintah ini menghapus modul yang bernama ContosoModule dari akun Otomatisasi yang bernama Contoso17.
