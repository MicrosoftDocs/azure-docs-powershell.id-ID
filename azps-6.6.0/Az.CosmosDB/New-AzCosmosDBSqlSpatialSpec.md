# New-AzCosmosDBSqlSpatialSpec

## SYNOPSIS
Membuat objek baru tipe PSSpatialSpec. Pernyataan ini bisa disampaikan sebagai nilai parameter untuk Set-AzCosmosDBSqlIndexingPolicy.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.cosmosdb/new-azcosmosdbsqlspatialspec) untuk informasi terkini.

## SYNTAX

```
New-AzCosmosDBSqlSpatialSpec -Path <String> -Type <String[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Membuat Objek terkait Jenis Spasial API Sql.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> New-AzCosmosDBSqlSpatialSpec -Path "/abc" -Type String
Path Types
---- -----
/abc {String}
```
