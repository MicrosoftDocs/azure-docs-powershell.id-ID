# New-AzCosmosDBSqlUniqueKey

## SYNOPSIS
Membuat objek CosmosDB Sql UniqueKey baru.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.cosmosdb/new-azcosmosdbsqluniquekey) untuk informasi terbaru.

## SYNTAX

```
New-AzCosmosDBSqlUniqueKey -Path <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzCosmosDBSqlUniqueKey** menciptakan objek baru tipe PSUniqueKey.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> New-AzCosmosDBSqlUniqueKey -Path {path}

Path
----
{path}
```
