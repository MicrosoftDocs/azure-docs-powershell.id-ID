# New-AzADGroup

## SYNOPSIS
Membuat grup direktori aktif baru.

[!INCLUDE [msgraph-migration-banner](../../includes/msgraph-migration-banner.md)]

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.resources/new-azadgroup) untuk informasi terbaru.

## SYNTAX

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Membuat grup direktori aktif baru. Berikut adalah izin yang diperlukan:

- Azure Active Directory Graph
  - Directory.ReadWrite.All
- Microsoft Graph
  - Directory.ReadWrite.All
  - PrivilegedAccess.ReadWrite.AzureADGroup

## EXAMPLES

### Contoh 1: Membuat grup AD baru

```powershell
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "MyGroupNick"
```

Membuat grup AD baru dengan nama "MyGroupDisplayName" dan nama panggilan email "MyGroupNick".
