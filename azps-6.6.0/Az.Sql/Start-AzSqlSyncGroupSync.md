# Start-AzSqlSyncGroupSync

## SYNOPSIS
Memulai sinkronisasi grup sinkronisasi.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.sql/start-azsqlsyncgroupsync) untuk informasi terbaru.

## SYNTAX

```
Start-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Start-AzSqlSyncGroupSync** memulai sinkronisasi grup sinkronisasi.

## EXAMPLES

### Contoh 1
```
PS C:\> Start-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Perintah ini memulai putaran sinkronisasi untuk mysg grup sinkronisasi.
