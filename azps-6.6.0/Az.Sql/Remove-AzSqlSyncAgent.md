# Remove-AzSqlSyncAgent

## SYNOPSIS
Menghapus Agen Sinkronisasi Azure SQL Anda.

## SYNTAX

```
Remove-AzSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzSqlSyncAgent** menghapus Agen Sinkronisasi Azure SQL.

## EXAMPLES

### Contoh 1: Menghapus agen sinkronisasi
```
PS C:\>Remove-AzSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Perintah ini menghapus Agen Sinkronisasi Azure SQL bernama syncAgent01.
