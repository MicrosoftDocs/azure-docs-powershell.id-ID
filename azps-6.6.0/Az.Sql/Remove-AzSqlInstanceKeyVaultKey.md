# Remove-AzSqlInstanceKeyVaultKey

## SYNOPSIS
Menghapus kunci Key Vault dari SQL terkelola

## SYNTAX

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (Default)
```
Remove-AzSqlInstanceKeyVaultKey [-ResourceGroupName] <String> [-InstanceName] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultInputObjectParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-Instance] <AzureSqlManagedInstanceModel> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-InstanceResourceId] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzSqlInstanceKeyVaultKey akan menghapus kunci Key Vault dari Instans Terkelola yang ditentukan. Perhatikan SQL izin instans terkelola ke vault kunci tidak diubah. Untuk mengubah izin, gunakan Set-AzKeyVaultAccessPolicy. Perlu diketahui bahwa cmdlet ini tidak melakukan perubahan pada Key Vault. Untuk menghapus kunci dari Key Vault, gunakan Remove-AzureKeyVaultKey.
