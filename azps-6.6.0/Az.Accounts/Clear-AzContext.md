# Clear-AzContext

## SYNOPSIS
Hapus semua kredensial, akun, dan informasi langganan Azure.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.accounts/clear-azcontext) untuk informasi terkini.

## SYNTAX

```
Clear-AzContext [-PassThru] [-Force] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Hapus semua informasi Kredensial, akun, dan langganan Azure.

## EXAMPLES

### Contoh 1: Hapus konteks global
```powershell
PS C:\> Clear-AzContext -Scope CurrentUser
```

Hapus semua informasi akun, langganan, dan kredensial untuk sesi powershell apa pun.
