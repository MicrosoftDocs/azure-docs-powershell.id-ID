# Open-AzSurveyLink

## SYNOPSIS
Buka link survei di browser default.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.accounts/open-azsurveylink) untuk informasi terkini.

## SYNTAX

```
Open-AzSurveyLink [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Buka link survei di browser default.

## EXAMPLES

### Contoh 1
```
Open-AzSurveyLink
Opening the default browser to https://aka.ms/azpssurvey?Q_CHL=INTERCEPT
```
