# Test-AzKustoClusterPrincipalAssignmentNameAvailability

## SYNOPSIS
Memeriksa apakah nama penetapan pokok valid dan belum digunakan.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.kusto/test-azkustoclusterprincipalassignmentnameavailability) untuk informasi terbaru.

## SYNTAX

### CheckExpanded (Default)
```
Test-AzKustoClusterPrincipalAssignmentNameAvailability -ClusterName <String> -ResourceGroupName <String>
 -Name <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### CheckViaIdentityExpanded
```
Test-AzKustoClusterPrincipalAssignmentNameAvailability -InputObject <IKustoIdentity> -Name <String>
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Memeriksa apakah nama penetapan pokok valid dan belum digunakan.

## EXAMPLES

### Contoh 1: Periksa ketersediaan nama principalassignment kluster yang sedang digunakan
```powershell
PS C:\> Test-AzKustoClusterPrincipalAssignmentNameAvailability -ResourceGroupName "testrg" -ClusterName "testnewkustocluster" -Name "myprincipal" -Type "Microsoft.Kusto/clusters/principalAssignments" 

Message                                                                                                        Name            NameAvailable Reason
-------                                                                                                        ----            ------------- ------
PrincipalAssignment myprincipal already exists in cluster testnewkustocluster. Please select a different name. myprincipal   False
```

Perintah di atas mengembalikan apakah PrincipalAssignment bernama "myprincipal" ada di kluster "testnewkustocluster".

### Contoh 2: Periksa ketersediaan nama principalassignment cluster yang tidak digunakan
```powershell
PS C:\> Test-AzKustoClusterPrincipalAssignmentNameAvailability -ResourceGroupName "testrg" -ClusterName "testnewkustocluster" -Name "newprincipal" -Type "Microsoft.Kusto/clusters/principalAssignments"

Message Name                  NameAvailable Reason
------- ----                  ------------- ------
        availablekustocluster True
```

Perintah di atas mengembalikan apakah PrincipalAssignment bernama "newprincipal" ada di kluster "testnewkustocluster".
