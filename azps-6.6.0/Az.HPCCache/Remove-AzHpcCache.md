# Remove-AzHpcCache

## SYNOPSIS
Menghapus HPC Cache.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.hpccache/remove-azhpccache) untuk informasi terbaru.

## SYNTAX

```
Remove-AzHpcCache -ResourceGroupName <String> -Name <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzHpcCache** menghapus HPC Cache Azure.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Remove-AzHpcCache -ResourceGroupName testRG -CacheName testCache
```
