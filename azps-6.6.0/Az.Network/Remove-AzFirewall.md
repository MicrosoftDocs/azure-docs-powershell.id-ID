# Remove-AzFirewall

## SYNOPSIS
Menghapus Firewall.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.network/remove-azfirewall) untuk informasi terbaru.

## SYNTAX

```
Remove-AzFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzFirewall** menghapus Azure Firewall.

## EXAMPLES

### Contoh 1: Membuat dan menghapus Firewall
```powershell
New-AzFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Contoh ini akan membuat Firewall lalu menghapusnya. Untuk menyembunyikan perintah saat menghapus Firewall, gunakan bendera -Force.
