# Remove-AzRouteFilter

## SYNOPSIS
Menghapus filter rute.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.network/remove-azroutefilter) untuk informasi terbaru.

## SYNTAX

```
Remove-AzRouteFilter -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzRouteFilter** menghapus filter rute.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Remove-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
```

Perintah menghapus filter rute bernama RouteFilter01 dalam grup sumber daya bernama ResourceGroup01.
