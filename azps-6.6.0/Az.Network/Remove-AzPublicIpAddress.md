# Remove-AzPublicIpAddress

## SYNOPSIS
Menghapus alamat IP publik.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.network/remove-azpublicipaddress) untuk informasi terbaru.

## SYNTAX

```
Remove-AzPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzPublicIpAddress** menghapus alamat IP publik Azure.

## EXAMPLES

### 1: Menghapus sumber daya alamat IP publik
```
Remove-AzPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Perintah ini menghapus sumber daya alamat IP publik bernama $publicIpName dalam $rgName grup sumber daya.
