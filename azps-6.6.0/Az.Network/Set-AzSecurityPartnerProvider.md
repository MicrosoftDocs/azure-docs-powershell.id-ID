# Set-AzSecurityPartnerProvider

## SYNOPSIS
Menyimpan Azure SecurityPartnerProvider yang dimodifikasi.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.network/set-azsecuritypartnerprovider) untuk informasi terbaru.

## SYNTAX

```
Set-AzSecurityPartnerProvider -SecurityPartnerProvider <PSSecurityPartnerProvider> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzSecurityPartnerProvider** memperbarui Azure SecurityPartnerProvider

## EXAMPLES

### Contoh 1
```powershell
$securityPartnerProvider = Get-AzSecurityPartnerProvider -ResourceGroupName securityPartnerProviderRG -Name securityPartnerProvider
Set-AzSecurityPartnerProvider -SecurityPartnerProvider $securityPartnerProvider
```
