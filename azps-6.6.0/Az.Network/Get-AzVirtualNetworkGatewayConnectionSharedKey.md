# Get-AzVirtualNetworkGatewayConnectionSharedKey

## SYNOPSIS
Menampilkan kunci bersama yang digunakan untuk sambungan.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.network/get-azvirtualnetworkgatewayconnectionsharedkey) untuk informasi terbaru.

## SYNTAX

```
Get-AzVirtualNetworkGatewayConnectionSharedKey [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Menampilkan kunci bersama yang digunakan untuk sambungan.

## EXAMPLES

### Contoh 1
```
Get-AzVirtualNetworkGatewayConnectionSharedKey -Name 1 -ResourceGroupName P2SVPNGateway
xxxxxx
```
