# Remove-AzPrivateDnsZoneGroup

## SYNOPSIS
Menghapus grup zona DNS.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.network/remove-azprivatednszonegroup) untuk informasi terkini.

## SYNTAX

```
Remove-AzPrivateDnsZoneGroup -ResourceGroupName <String> -PrivateEndpointName <String> -Name <String> [-Force]
 [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzPrivateDnsZoneGroup** menghapus grup zona DNS. 

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Remove-AzPrivateDnsZoneGroup -ResourceGroupName "rg" -PrivateEndpointName "test-pr-endpoint" -name dnsgroup1 
```

Contoh di atas menghapus grup zona DNS bernama dnsgroup1 dari titik akhir test-pr-endpoint.
