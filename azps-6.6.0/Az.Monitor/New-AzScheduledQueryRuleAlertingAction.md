# New-AzScheduledQueryRuleAlertingAction

## SYNOPSIS
Membuat objek dari tipe Tindakan Peringatan

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.monitor/new-azscheduledqueryrulealertingaction) untuk informasi terbaru.

## SYNTAX

```
New-AzScheduledQueryRuleAlertingAction [-AznsAction <PSScheduledQueryRuleAznsAction>] -Severity <String>
 [-ThrottlingInMinutes <Int32>] -Trigger <PSScheduledQueryRuleTriggerCondition>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Membuat objek dari tipe Tindakan Peringatan Objek ini akan dialihkan ke perintah yang membuat Aturan Peringatan Log

## EXAMPLES

### Contoh 1
```powershell
PS C:\> $alertingAction = New-AzScheduledQueryRuleAlertingAction -AznsAction $aznsActionGroup -Severity "1" -Trigger $triggerCondition
```
