# Stop-AzDataMigrationService

## SYNOPSIS
Menghentikan contoh Azure Database Migration Service yang berada dalam status berjalan.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.datamigration/stop-azdatamigrationservice) untuk informasi terbaru.

## SYNTAX

### ComponentNameParameterSet (Default)
```
Stop-AzDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Stop-AzDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Stop-AzDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Stop-AzDataMigrationService menghentikan contoh Azure Database Migration Service yang sedang berjalan.

## EXAMPLES

### Contoh 1
```
PS C:\> Stop-AzDataMigrationService -ResourceGroupName MyResourceGroup -ServiceName TestService
```

Contoh di atas menghentikan contoh Azure Database Migration Service yang disebut TestService berdasarkan nama layanan yang dikirim sebagai parameter input

### Contoh 2
```
PS C:\> Stop-AzDataMigrationService -InputObject $TestService
```

Contoh di atas menghentikan contoh Azure Database Migration Service berdasarkan objek PSDataMigrationService yang dikirim sebagai parameter input.
