# Set-AzPortalDashboard

## SYNOPSIS
Membuat atau memperbarui Dasbor.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.portal/set-azportaldashboard) untuk informasi terbaru.

## SYNTAX

```
Set-AzPortalDashboard -Name <String> -ResourceGroupName <String> -DashboardPath <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Membuat atau memperbarui Dasbor.

## EXAMPLES

### Contoh 1: Memperbarui definisi dasbor menggunakan templat dasbor
```powershell
PS C:\> Set-AzPortalDashboard -DashboardPath .\resources\dash1-update.json -ResourceGroupName my-rg -DashboardName dashbase03

Location Name       Type
-------- ----       ----
eastasia dashbase03 Microsoft.Portal/dashboards
```

Memperbarui definisi dasbor menggunakan file templat dasbor.
