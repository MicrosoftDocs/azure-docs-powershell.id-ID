# Get-AzRedisCachePatchSchedule

## SYNOPSIS
Mendapat jadwal patch.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.rediscache/get-azrediscachepatchschedule) untuk informasi terbaru.

## SYNTAX

```
Get-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzRedisCachePatchSchedule** mendapatkan jadwal patch untuk cache di Azure Redis Cache.

## EXAMPLES

### Contoh 1: Dapatkan jadwal patch
```
PS C:\>Get-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Perintah ini mendapatkan jadwal patch dari singgahan bernama RedisCache06.
