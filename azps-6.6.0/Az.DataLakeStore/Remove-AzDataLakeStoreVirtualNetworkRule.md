# Remove-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Menghapus aturan jaringan virtual tertentu dalam Penyimpanan Data Lake tertentu.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.datalakestore/remove-azdatalakestorevirtualnetworkrule) untuk informasi terbaru.

## SYNTAX

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzDataLakeStoreVirtualNetworkRule** menghapus aturan jaringan virtual tertentu dalam Data Lake Store yang ditentukan.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

Menghapus aturan jaringan virtual "myVNET" dari akun "dls"
