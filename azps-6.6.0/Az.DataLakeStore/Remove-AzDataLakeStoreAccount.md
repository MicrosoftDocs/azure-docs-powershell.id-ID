# Remove-AzDataLakeStoreAccount

## SYNOPSIS
Menghapus akun Data Lake Store secara permanen.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.datalakestore/remove-azdatalakestoreaccount) untuk informasi terbaru.

## SYNTAX

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzDataLakeStoreAccount** menghapus akun Data Lake Store secara permanen.

## EXAMPLES

### Contoh 1: Menghapus akun Penyimpanan Data Lake
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Perintah ini menghapus akun bernama ContosoADL dari Data Lake Store.
