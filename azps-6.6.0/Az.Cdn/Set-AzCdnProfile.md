# Set-AzCdnProfile

## SYNOPSIS
Memperbarui profil CDN Anda.

## SYNTAX

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzCdnProfile** memperbarui profil Azure Content Delivery Network (CDN).
