# Set-AzCdnEndpoint

## SYNOPSIS
Memperbarui titik akhir CDN.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.cdn/set-azcdnendpoint) untuk informasi terbaru.

## SYNTAX

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
**Cmdlet Set-AzCdnEndpoint** memperbarui titik akhir Azure Content Delivery Network (CDN).
