# Split-AzReservation

## SYNOPSIS
Memisahkan `Reservation` .

## SYNTAX

### CommandLine (Default)
```
Split-AzReservation -ReservationOrderId <Guid> -ReservationId <Guid> -Quantity <Int32[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PipeObject
```
Split-AzReservation -Quantity <Int32[]> -Reservation <PSReservation> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Pisahkan `Reservation` menjadi dua s dengan distribusi kuantitas `Reservation` tertentu.

## EXAMPLES

### Contoh 1
```
PS C:\> Split-AzReservation -ReservationOrderId "00000000-ffff-ffff-0000-00000fffff" -ReservationId "11111111-1111-1111-1111-1111111111" -Quantity 2,3
```

Memisahkan yang `Reservation` ditentukan menjadi dua dengan kuantitas `Reservation` terkait
