# Get-AzDiskEncryptionSet

## SYNOPSIS
Dapatkan atau  list enkripsi disk.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.compute/get-azdiskencryptionset) untuk informasi terkini.

## SYNTAX

```
Get-AzDiskEncryptionSet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Dapatkan atau  list enkripsi disk.

## EXAMPLES

```powershell
PS C:\> Get-AzDiskEncryptionSet -ResourceGroupName rg1 -Name enc1;

Name              : enc1
Type              : Microsoft.Compute/diskEncryptionSets

Tags              : {}
```

Dapatkan kumpulan enkripsi disk 'enc1'
