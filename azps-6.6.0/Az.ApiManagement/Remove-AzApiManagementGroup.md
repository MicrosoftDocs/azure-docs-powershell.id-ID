# Remove-AzApiManagementGroup

## SYNOPSIS
Menghapus grup manajemen API yang sudah ada.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.apimanagement/remove-azapimanagementgroup) untuk informasi terbaru.

## SYNTAX

```
Remove-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzApiManagementGroup** menghapus grup manajemen API yang sudah ada.

## EXAMPLES

### Contoh 1: Menghapus grup manajemen yang sudah ada
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementGroup -Context $apimContext -GroupId "Group0001" -Force
```

Perintah ini menghapus grup manajemen yang sudah ada bernama Group0001 dan tidak meminta konfirmasi kepada pengguna.
