# Remove-AzIotHub

## SYNOPSIS
Menghapus IotHub.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.iothub/remove-aziothub) untuk informasi terbaru.

## SYNTAX

```
Remove-AzIotHub [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus IotHub.

## EXAMPLES

### Contoh 1 Menghapus IotHub
```
PS C:\> Remove-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Menghapus IotHub bernama "myiothub"
