# Invoke-AzIotHubConfigurationMetricsQuery

## SYNOPSIS
Memanggil kueri metrik konfigurasi perangkat IoT.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.iothub/invoke-aziothubconfigurationmetricsquery) untuk informasi terbaru.

## SYNTAX

### ResourceSet (Default)
```
Invoke-AzIotHubConfigurationMetricsQuery [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 -MetricName <String> [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubConfigurationMetricsQuery [-InputObject] <PSIotHub> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubConfigurationMetricsQuery [-ResourceId] <String> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Mengevaluasi metrik sistem atau kustom target yang ditentukan dalam konfigurasi perangkat IoT.
Terdapat metrik sistem yang telah ditentukan sebelumnya yang dihitung oleh Iot Hub dan tidak dapat dikustomisasi.
- "Ditargetkan" menentukan jumlah perangkat kembar yang cocok dengan kondisi target.
- "Diterapkan" menentukan jumlah perangkat kembar yang telah diubah oleh konfigurasi. 

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Invoke-AzIotHubConfigurationMetricsQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myConfig1" -MetricName "warningLimit"
```

Evaluasi metrik 'warningLimit' yang ditentukan kustom.

### Contoh 2
```powershell
PS C:\> Invoke-AzIotHubConfigMetric -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myConfig1" -MetricName "applied" -MetricType "system"
```

Mengevaluasi metrik 'diterapkan' sistem.
