# Remove-AzKustoAttachedDatabaseConfiguration

## SYNOPSIS
Menghapus konfigurasi database lampiran dengan nama yang diberikan.

## SYNTAX

### Hapus (Default)
```
Remove-AzKustoAttachedDatabaseConfiguration -ClusterName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzKustoAttachedDatabaseConfiguration -InputObject <IKustoIdentity> [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Menghapus konfigurasi database lampiran dengan nama yang diberikan.

## EXAMPLES

### Contoh 1: Delete an existing AttachedDatabaseConfiguration by name
```powershell
PS C:\> Remove-AzKustoAttachedDatabaseConfiguration -ResourceGroupName "testrg" -ClusterName "testnewkustoclusterf" -Name "myfollowerconfiguration"
```

Perintah di atas menghapus database pengikut yang ditentukan dalam AttachedDatabaseConfiguration "myfollowerconfiguration" dari kluster "testnewkustoclusterf".
