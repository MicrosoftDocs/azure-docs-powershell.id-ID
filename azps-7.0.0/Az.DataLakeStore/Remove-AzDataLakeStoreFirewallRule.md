# Remove-AzDataLakeStoreFirewallRule

## SYNOPSIS
Menghapus aturan firewall tertentu di Data Lake Store yang ditentukan.

## SYNTAX

```
Remove-AzDataLakeStoreFirewallRule [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzDataLakeStoreFirewallRule** menghapus aturan firewall tertentu di Data Lake Store yang ditentukan.

## EXAMPLES

### Contoh 1: Menghapus aturan firewall dari akun
```
PS C:\> Remove-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyFirewallRule
```

Menghapus aturan firewall "MyFirewallRule" dari akun "ContosoADL"
