# Az.DeviceProvisioningServices Module
## Deskripsi
Perintah untuk Mengelola Layanan Penyediaan Perangkat Azure IoT Hub.

### [Add-AzIotDeviceProvisioningServiceAccessPolicy](Add-AzIoTDeviceProvisioningServiceAccessPolicy.md)
