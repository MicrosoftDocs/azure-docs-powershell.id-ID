# Get-AzLoadBalancerProbeConfig

## SYNOPSIS
Mendapatkan konfigurasi configuration configuration for a load balancer.

## SYNTAX

```
Get-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzLoadBalancerProbeConfig** mendapatkan satu atau beberapa konfigurasi untuk penyeimbang muat.

## EXAMPLES

### Contoh 1: Mendapatkan konfigurasi konfigurasi pemuatan
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $slb
```

Perintah pertama mendapatkan penyeimbang muat bernama MyLoadBalancer, lalu menyimpannya di nilai $slb.
Perintah kedua mendapatkan konfigurasi konfigurasi konfigurasi konfigurasi konfigurasi terkait bernama MyProbe dari penyeimbang muat $slb.
