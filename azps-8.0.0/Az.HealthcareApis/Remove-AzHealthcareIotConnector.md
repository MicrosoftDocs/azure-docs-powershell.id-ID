# Remove-AzHealthcareIotConnector

## SYNOPSIS
Menghapus Konektor IoT.

## SYNTAX

### Hapus (Default)
```
Remove-AzHealthcareIotConnector -Name <String> -ResourceGroupName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzHealthcareIotConnector -InputObject <IHealthcareApisIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Menghapus Konektor IoT.

## EXAMPLES

### Contoh 1: Menghapus Konektor IoT.
```powershell
PS C:\> Remove-AzHealthcareIotConnector -Name azpsiotconnector -ResourceGroupName azps_test_group -WorkspaceName azpshcws

```

Menghapus Konektor IoT.

### Contoh 2: Menghapus Konektor IoT.
```powershell
PS C:\> Get-AzHealthcareIotConnector -Name azpsiotconnector -ResourceGroupName azps_test_group -WorkspaceName azpshcws | Remove-AzHealthcareIotConnector

```

Menghapus Konektor IoT.
