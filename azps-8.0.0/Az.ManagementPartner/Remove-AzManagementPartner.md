# Remove-AzManagementPartner

## SYNOPSIS
Hapus ID Microsoft Partner Network (MPN) dari pengguna atau perwakilan layanan yang diautentikasi saat ini.

## SYNTAX

```
Remove-AzManagementPartner [-PartnerId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Hapus ID Microsoft Partner Network (MPN) dari pengguna atau perwakilan layanan yang diautentikasi saat ini.

## EXAMPLES

### Contoh 1
```powershell
Remove-AzManagementPartner -PartnerId 123457 -PassThru
```

```output
true
```

Menghapus mitra manajemen
