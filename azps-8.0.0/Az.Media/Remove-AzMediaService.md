# Remove-AzMediaService

## SYNOPSIS
Menghapus layanan media.

## SYNTAX

```
Remove-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzMediaService** menghapus layanan media.

## EXAMPLES

### Contoh 1: Menghapus layanan media
```powershell
Remove-AzMediaService -ResourceGroupName "ResourceGroup001" -AccountName "MediaService0011"
```

Perintah ini menghapus layanan media bernama MediaService0011 dalam grup sumber daya bernama ResourceGroup001.
