# Enable-AzHDInsightMonitoring

## SYNOPSIS
Mengaktifkan integrasi log Azure Monitor Klasik pada kluster HDInsight dan log yang relevan akan dikirim ke ruang kerja pemantauan yang ditentukan selama pengaktifan.

## SYNTAX

```
Enable-AzHDInsightMonitoring [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Enable-AzHDInsightMonitoring** memungkinkan integrasi log Azure Monitor Klasik pada kluster HDInsight.

## EXAMPLES

### Contoh 1
```powershell
Enable-AzHDInsightMonitoring -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>
```

```output
True
```

Pemantauan akan diaktifkan pada kluster HDInsight dan log yang relevan akan dikirim ke ruang kerja pemantauan dengan id 1d364e89-bb71-4503-aa3d-a23535aea7bd

### Contoh 2
```powershell
Enable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>
```

```output
True
```

Pemantauan akan diaktifkan pada kluster HDInsight dan log yang relevan akan dikirim ke ruang kerja pemantauan dengan id 1d364e89-bb71-4503-aa3d-a23535aea7bd
