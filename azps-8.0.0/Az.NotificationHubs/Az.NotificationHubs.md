# Modul Az.NotificationHubs
## Deskripsi
Topik ini menampilkan topik bantuan untuk cmdlet Azure Notification Hub. Hub pemberitahuan digunakan untuk mengirim pemberitahuan push ke beberapa klien terlepas dari platform (iOS, Android, Windows Phone 8, Windows Store, dll.) yang digunakan oleh klien tersebut. Hub ini kira-kira setara dengan aplikasi individual: masing-masing aplikasi Anda biasanya akan memiliki hub pemberitahuannya sendiri. Hub pemberitahuan diatur dalam kontainer logis yang dikenal sebagai namespace, dan aturan otorisasi Tanda Tangan Akses Bersama (SAS) digunakan untuk mengelola akses ke hub dan namespace layanan. Semua elemen ini dapat dikelola dengan menggunakan cmdlet Notification Hub.
