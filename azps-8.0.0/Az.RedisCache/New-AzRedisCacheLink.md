# New-AzRedisCacheLink

## SYNOPSIS
Buat tautan replikasi geografis antara dua Redis Cache.

## SYNTAX

```
New-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Buat tautan replikasi geografis antara dua Redis Cache.

## EXAMPLES

### Contoh 1: Membuat tautan di antara dua cache
```powershell
New-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

```output
        PrimaryServerName   : mycache1
        SecondaryServerName : mycache2
        ProvisioningState   : Creating
```

Perintah ini membuat tautan replikasi geografis antara Redis Cache mycache1 dan mycache2.
