# Remove-AzIotHub

## SYNOPSIS
Menghapus IotHub.

## SYNTAX

```
Remove-AzIotHub [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus IotHub.

## EXAMPLES

### Contoh 1 Menghapus IotHub
```powershell
Remove-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Menghapus IotHub bernama "myiothub"
