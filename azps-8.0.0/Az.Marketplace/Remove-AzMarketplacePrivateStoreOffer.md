# Remove-AzMarketplacePrivateStoreOffer

## SYNOPSIS
Menghapus penawaran dari penyimpanan privat.

## SYNTAX

```
Remove-AzMarketplacePrivateStoreOffer -PrivateStoreId <String> -OfferId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Hapus penawaran dari penyimpanan privat yang dibuat dalam cakupan penyewa.

## EXAMPLES

### Contoh 1
```powershell
Remove-AzMarketplacePrivateStoreOffer -privateStoreId 7gh67884-1r56-44fb-a93d-030d4ae08b2d -offerId  publisherid.offerid
```

Hapus penawaran dari penyimpanan privat yang dibuat dalam cakupan penyewa.
