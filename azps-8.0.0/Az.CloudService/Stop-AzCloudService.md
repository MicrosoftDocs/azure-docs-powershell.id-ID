# Stop-AzCloudService

## SYNOPSIS
Matikan layanan awan.
Perhatikan bahwa sumber daya masih terlampir dan Anda dikenakan biaya untuk sumber daya tersebut.

## SYNTAX

### PowerOff (Default)
```
Stop-AzCloudService -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### PowerOffViaIdentity
```
Stop-AzCloudService -InputObject <ICloudServiceIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Matikan layanan awan.
Perhatikan bahwa sumber daya masih terlampir dan Anda dikenakan biaya untuk sumber daya tersebut.

## EXAMPLES

### Contoh 1: Hentikan layanan cloud
```powershell
Stop-AzCloudService -ResourceGroupName "ContosOrg" -CloudServiceName "ContosoCS"
```

Perintah ini menghentikan semua instans peran milik layanan cloud bernama ContosoCS.
