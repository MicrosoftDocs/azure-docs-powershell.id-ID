# Stop-AzSpringCloudAppDeployment

## SYNOPSIS
Hentikan penyebaran.

## SYNTAX

### Hentikan (Default)
```
Stop-AzSpringCloudAppDeployment -AppName <String> -Name <String> -ResourceGroupName <String>
 -ServiceName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### StopViaIdentity
```
Stop-AzSpringCloudAppDeployment -InputObject <ISpringCloudIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Hentikan penyebaran.

## EXAMPLES

### Contoh 1: Hentikan Layanan Spring Cloud berdasarkan nama.
```powershell
Stop-AzSpringCloudAppDeployment -ResourceGroupName spring-cloud-rg -ServiceName spring-cloud-service -AppName gateway -DeploymentName default
```

Hentikan Spring Cloud Service berdasarkan nama.

### Contoh 2: Hentikan Spring Cloud Service dari pipa.
```powershell
Get-AzSpringCloudAppDeployment -ResourceGroupName spring-cloud-rg -ServiceName spring-cloud-service -AppName gateway -DeploymentName default | Stop-AzSpringCloudAppDeployment
```

Hentikan Spring Cloud Service dari pipa.
