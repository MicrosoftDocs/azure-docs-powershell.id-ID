# Get-AzManagementGroupEntity

## SYNOPSIS
Mencantumkan semua Entitas di bawah Penyewa saat ini

## SYNTAX

```
Get-AzManagementGroupEntity [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzManagementGroupEntity** mendapatkan semua entitas (Grup Manajemen dan Langganan) di bawah penyewa saat ini.

## EXAMPLES

### Contoh 1: Mendapatkan semua Grup Manajemen
```powershell
Get-AzManagementGroupEntity
```

