# Get-AzCosmosDBCassandraKeyspaceThroughput

## SYNOPSIS
Mendapatkan nilai throughput dari Cassandra Keyspace.

## SYNTAX

### ByNameParameterSet (Default)
```
Get-AzCosmosDBCassandraKeyspaceThroughput -ResourceGroupName <String> -AccountName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBCassandraKeyspaceThroughput -Name <String> -InputObject <PSDatabaseAccountGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
**Cmdlet Get-AzCosmosDBCassandraKeyspaceThroughput** mendapatkan objek throughput yang sesuai dengan Keyspace tertentu.

## EXAMPLES

### Contoh 1
```powershell
Get-AzCosmosDBCassandraKeyspaceThroughput -ResourceGroupName "resourceGroupName" -AccountName "accountName" -Name "name"
```
