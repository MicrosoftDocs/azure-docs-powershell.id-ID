# Baru-AzCosmosDBSqlUniqueKey

## SYNOPSIS
Membuat objek CosmosDB Sql UniqueKey baru.

## SYNTAX

```
New-AzCosmosDBSqlUniqueKey -Path <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzCosmosDBSqlUniqueKey** membuat objek baru jenis PSUniqueKey.

## EXAMPLES

### Contoh 1
```powershell
New-AzCosmosDBSqlUniqueKey -Path "path"
```

```output
Path
----
{path}
```
