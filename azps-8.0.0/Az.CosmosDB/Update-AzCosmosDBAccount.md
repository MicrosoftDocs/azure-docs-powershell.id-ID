# Update-AzCosmosDBAccount

## SYNOPSIS
Memperbarui atribut akun CosmosDB.

## SYNTAX

## DESCRIPTION
Perbarui properti akun CosmosDB. Tidak dapat memperbarui Wilayah Akun secara simulasi dengan properti lain.

## EXAMPLES

### Contoh 1
```powershell
Update-AzCosmosDBAccount -ResourceGroupName resourceGroupName -Name accountName -DefaultConsistencyLevel "Strong" -EnableAutomaticFailover 1 -EnableMultipleWriteLocations 1 -EnableVirtualNetwork 1
```

```output
Kind                          : GlobalDocumentDB

```

Memperbarui DefaultConsistencyLevel ke "Strong", Enabled AutomaticFailover, Enabled MultipleWriteLocations, dan Enabled VirtualNetwork for CosmosDB Account dengan nama accountName.
