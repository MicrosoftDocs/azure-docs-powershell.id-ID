# Get-AzDisk

## SYNOPSIS
Mendapatkan properti disk Terkelola.

## SYNTAX

```
Get-AzDisk [[-ResourceGroupName] <String>] [[-DiskName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzDisk** mendapatkan properti disk Terkelola.

## EXAMPLES

### Contoh 1
```powershell
Get-AzDisk -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'
```

```output

```

Perintah ini mendapatkan properti disk bernama 'Disk01' di grup sumber daya 'ResourceGroup01'.
