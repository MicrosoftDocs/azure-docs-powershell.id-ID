# Set-AzEventHubConsumerGroup

## SYNOPSIS
Memperbarui grup konsumen Azure Event Hubs yang ditentukan.

## SYNTAX

```
Set-AzEventHubConsumerGroup [-ResourceGroupName] <String> [-Namespace] <String> [-EventHub] <String>
 [-Name] <String> [[-UserMetadata] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Set-AzEventHubConsumerGroup memperbarui grup konsumen Azure Event Hubs yang ditentukan.

## EXAMPLES

### Contoh 1
```powershell
Set-AzEventHubConsumerGroup -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName -EventHubName MyEventHubName -ConsumerGroupName MyConsumerGroupName -UserMetadata "Testing"
```

Mengatur metadata pengguna dari grup \`konsumen MyConsumerGroupName\` ke "Pengujian."
