# Get-AzFrontDoorCdnProfileResourceUsage

## SYNOPSIS
Memeriksa kuota dan penggunaan aktual titik akhir AzureFrontDoor di bawah profil CDN yang diberikan.

## SYNTAX

```
Get-AzFrontDoorCdnProfileResourceUsage -ProfileName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Memeriksa kuota dan penggunaan aktual titik akhir AzureFrontDoor di bawah profil CDN yang diberikan.
