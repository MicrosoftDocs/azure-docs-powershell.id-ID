# Get-AzCdnEdgeNode

## SYNOPSIS
Edgenodes adalah lokasi Point of Presence (POP) global yang digunakan untuk mengirimkan konten CDN kepada pengguna akhir.

## SYNTAX

```
Get-AzCdnEdgeNode [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Edgenodes adalah lokasi Point of Presence (POP) global yang digunakan untuk mengirimkan konten CDN kepada pengguna akhir.
