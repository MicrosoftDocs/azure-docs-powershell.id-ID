# Remove-AzAutomationVariable

## SYNOPSIS
Menghapus variabel Automation.

## SYNTAX

```
Remove-AzAutomationVariable [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzAutomationVariable** menghapus variabel dari Azure Automation.

## EXAMPLES

### Contoh 1: Menghapus variabel
```powershell
Remove-AzAutomationVariable -AutomationAccountName "Contoso17" -Name "StringVariable22" -ResourceGroupName "ResourceGroup01"
```

Perintah ini menghapus variabel bernama StringVariable22 di akun Automation bernama Contoso17.
