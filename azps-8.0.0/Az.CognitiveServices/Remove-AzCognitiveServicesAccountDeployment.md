# Remove-AzCognitiveServicesAccountDeployment

## SYNOPSIS
Menghapus Penyebaran dari akun Cognitive Services

## SYNTAX

### DefaultParameterSet (Default)
```
Remove-AzCognitiveServicesAccountDeployment [-ResourceGroupName] <String> [-AccountName] <String>
 [-Name] <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzCognitiveServicesAccountDeployment [-InputObject] <Deployment> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzCognitiveServicesAccountDeployment [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus Penyebaran dari akun Cognitive Services

## EXAMPLES

### Contoh 1
```powershell
Remove-AzCognitiveServicesAccountDeployment -ResourceGroupName cognitive-services-resource-group -AccountName resource-name -Name "plan"
```

Menghapus Penyebaran dari akun Cognitive Services
