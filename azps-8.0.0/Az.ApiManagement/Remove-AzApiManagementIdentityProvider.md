# Remove-AzApiManagementIdentityProvider

## SYNOPSIS
Menghapus Konfigurasi Penyedia Identitas yang ada.

## SYNTAX

```
Remove-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus Konfigurasi Penyedia Identitas yang ada.

## EXAMPLES

### Contoh 1: Menghapus pengaturan IdP Facebook dari layanan ApiManagement
```powershell
$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
Remove-AzApiManagementIdentityProvider -Context $apimContext -Type 'Facebook' -PassThru
```

Menghapus konfigurasi yang terkait dengan konfigurasi IdP Facebook.
