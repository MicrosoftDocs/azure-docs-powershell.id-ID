# Remove-AzApiManagementOpenIdConnectProvider

## SYNOPSIS
Menghapus penyedia OpenID Connect.

## SYNTAX

```
Remove-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> -OpenIdConnectProviderId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzApiManagementOpenIdConnectProvider** menghapus penyedia OpenID Connect untuk Azure API Management.

## EXAMPLES

### Contoh 1: Menghapus penyedia
```powershell
$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
Remove-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvider01" -PassThru
```

Perintah ini menghapus penyedia yang memiliki ID OICProvider01.
