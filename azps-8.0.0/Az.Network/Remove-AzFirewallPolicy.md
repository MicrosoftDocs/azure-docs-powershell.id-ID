# Remove-AzFirewallPolicy

## SYNOPSIS
Menghapus Kebijakan Azure Firewall

## SYNTAX

### RemoveByNameParameterSet
```
Remove-AzFirewallPolicy -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceIdParameterSet
```
Remove-AzFirewallPolicy [-Force] [-PassThru] [-AsJob] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObjectParameterSet
```
Remove-AzFirewallPolicy [-Force] [-PassThru] [-AsJob] -InputObject <PSAzureFirewallPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzFirewallPolicy** menghapus Kebijakan Azure Firewall.

## EXAMPLES

### Contoh 1
```powershell
Remove-AzFirewallPolicy -Name firewallpolicy -ResourceGroupName TestRg
```

Contoh ini menghapus kebijakan firewall bernama "firewallpolicy" di grup sumber daya "TestRg"

### Contoh 2
```powershell
Remove-AzFirewallPolicy -Name firewallpolicy -ResourceId "/subscriptions/12345/resourceGroups/TestRg/providers/Microsoft.Network/firewallpolicies/firewallPolicy1"
```

Contoh ini menghapus kebijakan firewall dengan Id.

### Contoh: 3
```powershell
Remove-AzFirewallPolicy -InputObject $fp
```

Contoh ini menghapus kebijakan firewall $fp
