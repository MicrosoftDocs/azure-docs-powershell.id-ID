# Set-AzDataLakeStoreItemPermission

## SYNOPSIS
Memodifikasi oktal izin file atau folder di Data Lake Store.

## SYNTAX

```
Set-AzDataLakeStoreItemPermission [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Permission] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzDataLakeStoreItemPermission** memodifikasi oktal izin file atau folder di Data Lake Store.

## EXAMPLES

### Contoh 1: Mengatur oktal izin untuk item
```powershell
Set-AzDataLakeStoreItemPermission -AccountName "ContosoADL" -Path "/file.txt" -Permission 0770
```

Perintah ini mengatur oktal izin untuk file ke 0770, yang diterjemahkan untuk menghapus bit lengket, mengatur izin baca/tulis/eksekusi untuk pemilik file, mengatur izin baca/tulis/jalankan untuk grup pemilik file, dan menghapus izin baca/tulis/eksekusi untuk yang lain.
