# New-AzureRmManagementPartner

## SYNOPSIS
Mengaitkan ID Jaringan Mitra Microsoft (MPN) ke pengguna atau prinsipal layanan yang diautentikasi saat ini.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmManagementPartner [-PartnerId] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Mengaitkan ID Jaringan Mitra Microsoft (MPN) ke pengguna atau prinsipal layanan yang diautentikasi saat ini.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> New-AzureRmManagementPartner -PartnerId 4977985
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```
