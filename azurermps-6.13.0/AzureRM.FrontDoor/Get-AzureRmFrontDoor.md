# Get-AzureRmFrontDoor

## SYNOPSIS
Dapatkan penyeimbang muat Pintu Depan

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmFrontDoorGet** mendapatkan semua Pintu Depan yang sudah ada dalam langganan saat ini yang cocok dengan informasi yang disediakan.
