# Remove-AzureRmWebApp

## SYNOPSIS
Menghapus Azure Web App.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### S1
```
Remove-AzureRmWebApp [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzureRmWebApp [-Force] [-AsJob] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmWebApp** menghapus Azure Web App yang disediakan grup sumber daya dan nama Web App.
Cmdlet ini, secara default, juga menghapus semua slot dan metrik.

## EXAMPLES

### Contoh 1: Menghapus Web App
```
PS C:\>Remove-AzureRmWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Perintah ini menghapus Azure Web App bernama ContosoSite yang dimiliki grup sumber daya bernama Default-Web-WestUS.
