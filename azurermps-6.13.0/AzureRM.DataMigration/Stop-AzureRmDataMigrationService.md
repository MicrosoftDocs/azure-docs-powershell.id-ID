# Stop-AzureRmDataMigrationService

## SYNOPSIS
Menghentikan contoh Azure Database Migration Service yang berada dalam status berjalan.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### ComponentNameParameterSet (Default)
```
Stop-AzureRmDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Stop-AzureRmDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Stop-AzureRmDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Stop-AzureRmDataMigrationService menghentikan contoh Azure Database Migration Service yang sedang berjalan.

## EXAMPLES

### Contoh 1
```
PS C:\> Stop-AzureRmDataMigrationService -ResourceGroupName MyResourceGroup -ServiceName TestService
```

Contoh di atas menghentikan contoh Azure Database Migration Service yang disebut TestService berdasarkan nama layanan yang dikirim sebagai parameter input

### Contoh 2
```
PS C:\> Stop-AzureRmDataMigrationService -InputObject $TestService
```

Contoh di atas menghentikan contoh Azure Database Migration Service berdasarkan objek PSDataMigrationService yang dikirim sebagai parameter input.
