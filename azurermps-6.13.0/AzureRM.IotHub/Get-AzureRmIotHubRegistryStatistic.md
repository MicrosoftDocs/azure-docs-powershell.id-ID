# Get-AzureRmIotHubRegistryStatistic

## SYNOPSIS
Mendapatkan RegistryStatistics untuk IotHub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmIotHubRegistryStatistic [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Mendapatkan RegistryStatistics untuk IotHub.
Ini menyediakan informasi tentang jumlah perangkat total, yang diaktifkan dan dinonaktifkan di IotHub.

## EXAMPLES

### Contoh 1 Dapatkan RegistryStatistics
```
PS C:\> Get-AzureRmIotHubRegistryStatistic -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Mendapatkan RegistryStatictics untuk IotHub bernama "myiothub"
