# Remove-AzureRmSnapshot

## SYNOPSIS
Menghapus snapshot.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSnapshot [-ResourceGroupName] <String> [-SnapshotName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmSnapshot** menghapus snapshot.

## EXAMPLES

### Contoh 1
```
PS C:\> Remove-AzureRmSnapshot -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01' -Force;
```

Perintah ini menghapus snapshot bernama 'Snapshot01' dalam grup sumber daya 'ResourceGroup01'.
