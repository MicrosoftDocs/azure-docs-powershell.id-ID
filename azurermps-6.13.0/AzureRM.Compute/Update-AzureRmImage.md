# Update-AzureRmImage

## SYNOPSIS
Memperbarui gambar.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Update-AzureRmImage [-ResourceGroupName] <String> [-ImageName] <String> [-Image] <PSImage> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Update-AzureRmImage** memperbarui gambar.

## EXAMPLES

### Contoh 1
```
PS C:\> Get-AzureRmImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' | Remove-AzureRmImageDataDisk -Lun 1 | Update-AzureRmImage;
```

Perintah ini menghapus disk data dari unit logika nomor 1 dari gambar yang sudah ada 'Image01' dalam grup sumber daya 'ResourceGroup01'.
