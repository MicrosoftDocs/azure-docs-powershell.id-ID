# Revoke-AzureRmDiskAccess

## SYNOPSIS
Mencabut akses ke diska.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Revoke-AzureRmDiskAccess [-ResourceGroupName] <String> [-DiskName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Revoke-AzureRmDiskAccess** mencabut akses ke disk.

## EXAMPLES

### Contoh 1
```
PS C:\> Revoke-AzureRmDiskAccess -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'
```

Mencabut akses ke disk bernama 'Disk01' dalam grup sumber daya bernama 'ResourceGroup01'
