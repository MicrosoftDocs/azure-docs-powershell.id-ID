# Get-AzureRmVmssVMDiskEncryption

## SYNOPSIS
Memperlihatkan status enkripsi disk VM dalam kumpulan skala VM.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmVmssVMDiskEncryption [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [[-InstanceId] <String>] [-ExtensionName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Memperlihatkan status enkripsi disk yang ditetapkan skala VM.

## EXAMPLES

### Contoh 1
```
PS C:\> Get-AzureRmVmssVMDiskEncryption -ResourceGroupName "Group001" -VMScaleSetName "VMSS001" -InstanceId "1"
```

Memperlihatkan status enkripsi disk VM instance 1 dalam skala VM yang bernama VMSS001 yang dimiliki oleh grup sumber daya bernama Group001.
