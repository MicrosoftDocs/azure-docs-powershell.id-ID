# Revoke-AzureRmSnapshotAccess

## SYNOPSIS
Mencabut akses ke snapshot.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Revoke-AzureRmSnapshotAccess [-ResourceGroupName] <String> [-SnapshotName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Revoke-AzureRmSnapshotAccess** mencabut akses ke snapshot.

## EXAMPLES

### Contoh 1
```
PS C:\> Revoke-AzureRmDiskAccess -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01'
```

Mencabut akses ke snapshot yang bernama 'Snapshot01' dalam grup sumber daya bernama 'ResourceGroup01'
