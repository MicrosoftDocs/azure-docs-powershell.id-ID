# Get-AzureRmVMImageOffer

## SYNOPSIS
Mendapatkan tipe penawaran VMImage.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmVMImageOffer -Location <String> -PublisherName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmVMImageOffer** mendapatkan tipe penawaran VMImage.

## EXAMPLES

### Contoh 1: Mendapatkan tipe penawaran untuk penerbit
```
PS C:\> Get-AzureRmVMImageOffer -Location "Central US" -PublisherName "Fabrikam"
```

Perintah ini mendapatkan tipe penawaran untuk penerbit tertentu di kawasan AS Tengah.
