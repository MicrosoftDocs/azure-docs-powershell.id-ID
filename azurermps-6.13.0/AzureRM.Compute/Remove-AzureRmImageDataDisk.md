# Remove-AzureRmImageDataDisk

## SYNOPSIS
Menghapus disk data dari objek gambar.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmImageDataDisk [-Image] <PSImage> [-Lun] <Int32> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmImageDataDisk** menghapus disk data dari objek gambar.

## EXAMPLES

### Contoh 1
```
PS C:\> Get-AzureRmImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' | Remove-AzureRmImageDataDisk -Lun 1 | Update-AzureRmImage;
```

Perintah ini menghapus disk data unit logika nomor 1 dari gambar 'Image01' yang sudah ada dalam grup sumber daya 'ResourceGroup01'.
