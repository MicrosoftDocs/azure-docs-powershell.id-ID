# Remove-AzureRmMediaService

## SYNOPSIS
Menghapus layanan media.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmMediaService** menghapus layanan media.

## EXAMPLES

### Contoh 1: Menghapus layanan media
```
PS C:\>Remove-AzureRmMediaService -ResourceGroupName "ResourceGroup001" -AccountName "MediaService0011"
```

Perintah ini menghapus layanan media bernama MediaService0011 dalam grup sumber daya bernama ResourceGroup001.
