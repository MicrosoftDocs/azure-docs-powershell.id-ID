# New-AzureRmNetworkSecurityRuleConfig

## SYNOPSIS
Membuat konfigurasi aturan keamanan jaringan.

## DESCRIPTION
Cmdlet **New-AzureRmNetworkSecurityRuleConfig** membuat konfigurasi aturan keamanan jaringan Azure untuk grup keamanan jaringan.

## EXAMPLES

### 1: Membuat aturan keamanan jaringan untuk memperbolehkan RDP
```
$rule1 = New-AzureRmNetworkSecurityRuleConfig -Name rdp-rule -Description "Allow RDP" 
    -Access Allow -Protocol Tcp -Direction Inbound -Priority 100 -SourceAddressPrefix 
    Internet -SourcePortRange * -DestinationAddressPrefix * -DestinationPortRange 3389
```

Perintah ini membuat aturan keamanan yang memungkinkan akses dari Internet ke port 3389

### 2: Membuat aturan keamanan jaringan yang memungkinkan HTTP
```
$rule2 = New-AzureRmNetworkSecurityRuleConfig -Name web-rule -Description "Allow HTTP" 
    -Access Allow -Protocol Tcp -Direction Inbound -Priority 101 -SourceAddressPrefix 
    Internet -SourcePortRange * -DestinationAddressPrefix * -DestinationPortRange 80
```

Perintah ini membuat aturan keamanan yang memungkinkan akses dari Internet ke port 80
