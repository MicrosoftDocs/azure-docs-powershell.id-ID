# Get-AzureRmLocalNetworkGateway

## SYNOPSIS
Mendapatkan Gateway Jaringan Lokal

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmLocalNetworkGateway [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Gateway Jaringan Lokal adalah objek yang mewakili perangkat VPN Lokal Anda.
Cmdlet **Get-AzureRmLocalNetworkGateway** mengembalikan objek yang mewakili gateway lokal Anda berdasarkan Nama dan Nama Grup Sumber Daya.

## EXAMPLES

### 1: Dapatkan Gateway Jaringan Lokal
```
Get-AzureRmLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Mengembalikan objek Local Network Gateway dengan nama "myLocalGW" di dalam grup sumber daya "myRG"
