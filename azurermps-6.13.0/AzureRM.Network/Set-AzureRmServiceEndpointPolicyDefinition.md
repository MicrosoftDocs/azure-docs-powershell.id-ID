## SYNTAX

```
Set-AzureRmServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <System.Collections.Generic.List`1[System.String]>]
 [-Service <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzureRmServiceEndpointPolicyDefinition** membuat definisi kebijakan titik akhir layanan.

## EXAMPLES

### Contoh 1: Memperbarui definisi kebijakan titik akhir layanan dalam kebijakan titik akhir layanan
```
$serviceEndpointPolicy = Set-AzureRmServiceEndpointPolicyDefinition -Name "Policydef1" -ServiceEndpointPolicy $serviceEndpointPolicy
```

Perintah ini memperbarui definisi kebijakan titik akhir layanan yang bernama Policydef1 dalam kebijakan titik akhir layanan yang ditentukan oleh $ServiceEndpointPolicy.
