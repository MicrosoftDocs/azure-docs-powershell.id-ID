# Set-AzureRmVirtualNetworkTap

## SYNOPSIS
Mengatur status tujuan untuk ketuk jaringan virtual.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
**Set-AzureRmVirtualNetworkTap** mengatur status tujuan untuk ketuk jaringan virtual Azure.

## EXAMPLES

### Contoh 1: Mengonfigurasi ketuk Jaringan virtual
```
PS C:\>$vTap = Get-AzureRmVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzureRmVirtualNetworkTap -VirtualNetworkTap $vTap
```

Perintah memperbarui IpConfiguration Tujuan dan memperbarui ketuk Jaringan virtual.
Jika ada konfigurasi ketukan yang mereferensikannya, maka semua lalu lintas sumber tidak akan mulai dicerminkan ke pembaruan postingan konfigurasi ip tujuan baru.
