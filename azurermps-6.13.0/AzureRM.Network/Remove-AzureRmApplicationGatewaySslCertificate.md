# Remove-AzureRmApplicationGatewaySslCertificate

## SYNOPSIS
Menghapus sertifikat SSL dari gateway aplikasi Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmApplicationGatewaySslCertificate** menghapus sertifikat Lapisan Soket Aman (SSL) dari gateway aplikasi Azure.

## EXAMPLES

### Contoh 1: Menghapus sertifikat SSL dari gateway aplikasi
```
PS C:\>$AppGW = Get-AzureRmApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzureRmApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

Perintah pertama mendapatkan gateway aplikasi bernama ApplicationGateway01 dan menyimpan hasilnya dalam variabel yang bernama $AppGW.
Perintah kedua menghapus sertifikat SSL yang bernama Cert02 dari gateway aplikasi yang disimpan di $AppGW ini.
