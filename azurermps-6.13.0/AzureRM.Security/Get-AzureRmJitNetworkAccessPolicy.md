# Get-AzureRmJitNetworkAccessPolicy

## SYNOPSIS
Mendapatkan kebijakan akses jaringan JIT

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### SubscriptionScope (Default)
```
Get-AzureRmJitNetworkAccessPolicy [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupScope
```
Get-AzureRmJitNetworkAccessPolicy -ResourceGroupName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceGroupLevelResource
```
Get-AzureRmJitNetworkAccessPolicy -ResourceGroupName <String> -Location <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzureRmJitNetworkAccessPolicy -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Kebijakan akses jaringan Just In Time (JIT) memungkinkan Anda menetapkan kebijakan, yang akan memungkinkan pengguna sederhana untuk membuat koneksi jaringan sementara ke VM.
Kebijakan menentukan port, protokol, dan alamat IP sumber yang dapat meminta koneksi ke VM dan durasi maksimum sebelum koneksi akan ditutup secara otomatis.
Dalam kebijakan tersebut Anda juga bisa melihat permintaan koneksi yang dibuat dengan kebijakan ini.
