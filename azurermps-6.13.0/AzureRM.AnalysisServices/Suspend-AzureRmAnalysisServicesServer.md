# Suspend-AzureRmAnalysisServicesServer

## SYNOPSIS
Menangguhkan contoh server Analysis Services

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Suspend-AzureRmAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Suspend-AzureRmAnalysisServicesServer menangguhkan contoh server Analysis Services

## EXAMPLES

### Contoh 1
```
PS C:\> Suspend-AzureRmAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Perintah ini akan menangguhkan server aktif bernama testserver dalam grup uji grup sumber daya
