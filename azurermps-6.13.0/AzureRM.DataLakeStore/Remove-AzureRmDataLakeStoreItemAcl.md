# Remove-AzureRmDataLakeStoreItemAcl

## SYNOPSIS
Menghapus ACL file atau folder di Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmDataLakeStoreItemAcl** membersihkan daftar kontrol akses (ACL, Access Control List) dari file atau folder di Data Lake Store.

## EXAMPLES

### Contoh 1: Menghapus ACL dari folder
```
PS C:\>Remove-AzureRmDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Perintah ini menghapus ACL untuk direktori akar untuk akun ContosoADL.
