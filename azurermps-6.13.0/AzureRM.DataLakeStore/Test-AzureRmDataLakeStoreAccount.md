# Test-AzureRmDataLakeStoreAccount

## SYNOPSIS
Menguji keberadaan akun Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Test-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Test-AzureRmDataLakeStoreAccount** menguji keberadaan akun Data Lake Store.

## EXAMPLES

### Contoh 1: Menguji akun
```
PS C:\>Test-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

Perintah ini menguji apakah akun yang bernama ContosoADL ada.
