# Get-AzureRmConsumptionReservationSummary

## SYNOPSIS
Dapatkan ringkasan reservasi untuk penggunaan harian atau bulanan.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmConsumptionReservationSummary -Grain <String> -ReservationOrderId <String> [-ReservationId <String>]
 [-StartDate <DateTime>] [-EndDate <DateTime>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmConsumptionReservationSum cmdlet** mendapatkan ringkasan reservasi untuk penggunaan harian atau bulanan.
