# Revoke-AzureRmHDInsightRdpServicesAccess

## SYNOPSIS
Menonaktifkan akses RDP ke klaster Windows.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Revoke-AzureRmHDInsightRdpServicesAccess [-ClusterName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Revoke-AzureRmHDInsightRdpServicesAccess** menonaktifkan akses Remote Desktop Protocol (RDP) ke kluster Azure HDInsight berbasis Windows.

## EXAMPLES

### Contoh 1: Menonaktifkan akses RDP ke kluster tertentu
```
PS C:\>Revoke-AzureRmHDInsightRdpServicesAccess -ClusterName "your-hadoop-001"
```

Perintah ini mencabut akses RDP ke kluster bernama your-hadoop-001.
