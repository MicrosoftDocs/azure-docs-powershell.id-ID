# Grant-AzureRmHDInsightHttpServicesAccess

## SYNOPSIS
Memberikan akses HTTP ke kluster.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Grant-AzureRmHDInsightHttpServicesAccess [-ClusterName] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Grant-AzureRmHDInsightHttpServicesAccess** memberikan akses HTTP ke kluster Azure HDInsight menggunakan layanan ODBC, Ambari, Oozie, dan web.

## EXAMPLES

### Contoh 1: Memberikan akses HTTP ke kluster Azure HDInsight
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzureRmHDInsightHttpServicesAccess `
            -ClusterName $clusterName `
            -HttpCredential $newClusterCreds
```

Perintah ini memberikan akses HTTP ke kluster bernama your-hadoop-001.
