# Remove-AzureRmAutomationSourceControl

## SYNOPSIS
Menghapus kontrol sumber Azure Automation.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzureRmAutomationSourceControl menghapus kontrol sumber dari Azure Automation.

## EXAMPLES

### Contoh 1
Perintah ini menghapus kontrol sumber Otomatisasi bernama VSTSNative dalam akun bernama devAccount.
Perintah ini menentukan parameter *Paksa* . Oleh karena itu, tindakan ini tidak meminta konfirmasi kepada Anda.

```powershell
PS C:\> Remove-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
