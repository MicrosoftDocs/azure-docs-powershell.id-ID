# Remove-AzureRmAutomationSchedule

## SYNOPSIS
Menghapus jadwal Otomatisasi.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmAutomationSchedule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmAutomationSchedule** menghapus jadwal dari Azure Automation.

## EXAMPLES

### Contoh 1: Menghapus jadwal
```
PS C:\>Remove-AzureRmAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -ResourceGroupName "ResourceGroup01"
```

Perintah ini menghapus jadwal bernama Schedule01 dalam akun otomatisasi Contoso17 dalam grup sumber daya ResourceGroup01.
