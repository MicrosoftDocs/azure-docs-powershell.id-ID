# Stop-AzureRmAutomationJob

## SYNOPSIS
Menghentikan pekerjaan Otomatisasi.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Stop-AzureRmAutomationJob** menghentikan pekerjaan Azure Automation.
Tentukan tugas Otomatisasi yang berjalan.

## EXAMPLES

### Contoh 1: Menghentikan pekerjaan
```
PS C:\>Stop-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Perintah ini menghentikan pekerjaan yang memiliki ID yang ditentukan.
