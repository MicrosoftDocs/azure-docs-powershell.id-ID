# Get-AzureRmAutomationSourceControlSyncJobOutput

## SYNOPSIS
Mendapatkan output pekerjaan sinkronisasi kontrol sumber Otomatisasi Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmAutomationSourceControlSyncJobOutput -SourceControlName <String> -JobId <Guid>
 [-Stream <SourceControlSyncJobStreamType>] [-StreamId <String>] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmAutomationSourceControlSyncJobOutput** mendapatkan output untuk pekerjaan sinkronisasi kontrol sumber Otomatisasi Azure.

## EXAMPLES

### Contoh 1
Perintah ini mendapatkan output pekerjaan sinkronisasi kontrol sumber dengan id 08d6d266-27b6-463c-beea-bc48a67ace15 untuk kontrol sumber VSTSNative.

PS C:\> Get-AzureRmAutomationSourceControlSyncJobOutput -ResourceGroupName "rg1" `

                                                        -Stream Output | ForEach-Object {$_.summary}
