# Remove-AzureRmSqlServerThreatDetectionPolicy

## SYNOPSIS
Menghapus kebijakan deteksi ancaman dari server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSqlServerThreatDetectionPolicy [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzureRmSqlServerThreatDetectionPolicy menghapus kebijakan deteksi ancaman dari server Azure SQL.
Untuk menggunakan cmdlet ini, tentukan parameter ResourceGroupName dan ServerName untuk mengidentifikasi server tempat cmdlet ini menghapus kebijakan.

## EXAMPLES

### Contoh 1: Menghapus kebijakan deteksi ancaman untuk database
```
PS C:\> Remove-AzureRmSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Perintah ini menghapus kebijakan deteksi ancaman dari server bernama Server01.
