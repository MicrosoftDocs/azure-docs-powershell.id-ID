# Get-AzureRmSqlElasticPoolAdvisor

## SYNOPSIS
Mendapatkan satu atau beberapa Penasihat untuk Azure SQL Elastic Pool.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlElasticPoolAdvisor** mendapatkan satu atau beberapa Azure SQL Elastic Pool Advisors untuk Azure SQL Elastic Pool.
