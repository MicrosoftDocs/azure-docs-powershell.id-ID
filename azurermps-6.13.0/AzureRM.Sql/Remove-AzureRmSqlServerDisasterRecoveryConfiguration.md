# Remove-AzureRmSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Menghapus konfigurasi pemulihan sistem server database SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmSqlServerDisasterRecoveryConfiguration** menghapus konfigurasi pemulihan sistem server database SQL.
