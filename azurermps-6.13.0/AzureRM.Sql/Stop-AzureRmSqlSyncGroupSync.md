# Stop-AzureRmSqlSyncGroupSync

## SYNOPSIS
Menghentikan sinkronisasi grup sinkronisasi.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureRmSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Stop-AzureRmSqlSyncGroupSync** menghentikan sinkronisasi grup sinkronisasi.

## EXAMPLES

### Contoh 1
```
PS C:\> Stop-AzureRmSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Perintah ini menghentikan sinkronisasi yang sedang berlangsung untuk grup sinkronisasi mysg.
