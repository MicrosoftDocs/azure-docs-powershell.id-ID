# New-AzureRmSqlServerDnsAlias

## SYNOPSIS
Perintah ini akan membuat Alias DNS Azure SQL Server baru.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Membuat nama Alias DNS Azure SQL Server baru yang menunjuk ke server tertentu.

## EXAMPLES

### Contoh 1
```
PS C:\> $serverDNSAlias = New-AzureRmSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Perintah ini membuat Azure SQL Server Alias DNS dengan nama aliasName yang menunjuk ke serverName
