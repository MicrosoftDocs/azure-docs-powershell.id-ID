# Get-AzureRmSqlDatabaseThreatDetectionPolicy

## SYNOPSIS
Mendapatkan kebijakan deteksi ancaman untuk database.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlDatabaseThreatDetectionPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlDatabaseThreatDetectionPolicy** mendapatkan kebijakan deteksi ancaman dari database SQL Azure.
Untuk menggunakan cmdlet ini, tentukan parameter *ResourceGroupName*, *ServerName*, dan *DatabaseName* untuk mengidentifikasi database di mana cmdlet ini mendapatkan kebijakan.

## EXAMPLES

### Contoh 1: Mendapatkan kebijakan deteksi ancaman untuk database
```
PS C:\>Get-AzureRmSqlDatabaseThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
DatabaseName                 : Database01
ResourceGroupName            : ResourceGroup11
ServerName                   : Server01
ThreatDetectionState         : Enabled
NotificationRecipientsEmails : admin@myCompany.com
StorageAccountName           : mystorage
EmailAdmins                  : True
ExcludedDetectionTypes       : {}
RetentionInDays              : 0
```

Perintah ini mendapatkan kebijakan deteksi ancaman untuk database bernama Database01.
Database berada di server bernama Server01, yang ditetapkan untuk grup sumber daya ResourceGroup11.
