# Get-AzureRmSqlInstance

## SYNOPSIS
Mengembalikan informasi tentang Instans Database Terkelola Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### GetInstanceByResourceGroup (Default)
```
Get-AzureRmSqlInstance [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetInstanceByNameAndResourceGroup
```
Get-AzureRmSqlInstance [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlInstance** mengembalikan informasi tentang satu atau beberapa instans terkelola Azure SQL.
Tentukan nama instans untuk melihat informasi hanya untuk instans tersebut.
