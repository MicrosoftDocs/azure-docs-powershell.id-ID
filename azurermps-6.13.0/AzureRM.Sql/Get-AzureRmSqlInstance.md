# Get-AzureRmSqlInstance

## SYNOPSIS
Mengembalikan informasi tentang Azure SQL Contoh Database Terkelola.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### GetInstanceByResourceGroup (Default)
```
Get-AzureRmSqlInstance [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetInstanceByNameAndResourceGroup
```
Get-AzureRmSqlInstance [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlInstance** mengembalikan informasi tentang satu atau beberapa Azure SQL Managed Instance.
Tentukan nama instans untuk melihat informasi hanya untuk contoh tersebut.
