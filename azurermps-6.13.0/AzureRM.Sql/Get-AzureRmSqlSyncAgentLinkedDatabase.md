# Get-AzureRmSqlSyncAgentLinkedDatabase

## SYNOPSIS
Mengembalikan informasi tentang SQL Server database yang ditautkan oleh agen sinkronisasi.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlSyncAgentLinkedDatabase [-ServerName] <String> [-SyncAgentName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlSyncAgentLinkedDatabases** mengembalikan informasi tentang database SQL Server ditautkan oleh agen sinkronisasi.

## EXAMPLES

### Contoh 1: Dapatkan database SQL Server tertaut untuk agen sinkronisasi Azure SQL.
```
PS C:\> Get-AzureRmSqlSyncAgentLinkedDatabases -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01" | Format-List
SeverName                 : sever01
DatabaseId                : databaseId
DatabaseName              : database01
DatabaseType              : SQLServerDatabase
Description               : 
UserName                  : myAccount
```

Perintah ini mengembalikan database SQL Server yang ditautkan yang ditautkan oleh agen sinkronisasi Azure SQL.
