# Get-AzureRmSqlServerThreatDetectionPolicy

## SYNOPSIS
Mendapatkan kebijakan deteksi ancaman untuk server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlServerThreatDetectionPolicy -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlServerThreatDetectionPolicy** mendapatkan kebijakan deteksi ancaman server Azure SQL.
Untuk menggunakan cmdlet ini, tentukan parameter *ResourceGroupName* dan *ServerName* untuk mengidentifikasi server tempat cmdlet ini mendapatkan kebijakan.

## EXAMPLES

### Contoh 1: Dapatkan kebijakan deteksi ancaman untuk server
```
PS C:\>Get-AzureRmSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
ResourceGroupName            : ResourceGroup11
ServerName                   : Server01
ThreatDetectionState         : Enabled
NotificationRecipientsEmails : admin@myCompany.com
StorageAccountName           : mystorage
EmailAdmins                  : True
ExcludedDetectionTypes       : {}
RetentionInDays              : 0
```

Perintah ini mendapatkan kebijakan deteksi ancaman untuk server bernama Server01.
Server ditetapkan ke grup sumber daya ResourceGroup11.
