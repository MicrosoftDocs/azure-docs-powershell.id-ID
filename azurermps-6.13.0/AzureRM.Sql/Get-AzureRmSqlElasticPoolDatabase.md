# Get-AzureRmSqlElasticPoolDatabase

## SYNOPSIS
Mendapatkan database elastis dalam kumpulan elastis dan nilai propertinya.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlElasticPoolDatabase** mendapatkan database elastis dalam kumpulan elastis dan nilai propertinya.
Anda dapat menentukan nama database elastis dalam Azure SQL Database untuk melihat nilai properti hanya untuk database tersebut.

## EXAMPLES

### Contoh 1: Dapatkan semua database dalam kumpulan elastis
```
PS C:\>Get-AzureRmSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Perintah ini mendapatkan semua database dalam kumpulan elastis bernama ElasticPool01.
