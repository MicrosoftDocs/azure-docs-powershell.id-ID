# Set-AzureRmSqlServerRecommendedActionState

## SYNOPSIS
Memperbarui status tindakan azure SQL Server disarankan.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzureRmSqlServerRecommendedActionState** memperbarui status tindakan Azure SQL Server yang disarankan.
Cmdlet ini menerapkan, mengembalikan, atau membuang tindakan yang direkomendasikan berdasarkan status baru.
