# Test-AzureRmDataLakeAnalyticsAccount

## SYNOPSIS
Memeriksa keberadaan akun Analitik Danau Data.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Test-AzureRmDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Test-AzureRmDataLakeAnalyticsAccount** memeriksa keberadaan akun Analitik Danau Data.

## EXAMPLES

### Contoh 1: Menguji apakah ada akun
```
PS C:\>Test-AzureRmDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Perintah ini menguji apakah akun yang bernama ContosoAdlAccount sudah ada.
