# Wait-AzureRmDataLakeAnalyticsJob

## SYNOPSIS
Menunggu pekerjaan selesai.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Wait-AzureRmDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [[-WaitIntervalInSeconds] <Int32>]
 [[-TimeoutInSeconds] <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Wait-AzureRmDataLakeAnalyticsJob** menunggu pekerjaan Azure Data Lake Analytics selesai.

## EXAMPLES

### Contoh 1: Tunggu hingga pekerjaan selesai
```
PS C:\>Wait-AzureRmDataLakeAnalyticsJob -Account "ContosoAdlAccount" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Perintah berikut ini menunggu pekerjaan dengan ID yang ditentukan selesai.
