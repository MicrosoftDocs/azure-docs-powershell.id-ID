# Get-AzureRmRelayHybridConnection

## SYNOPSIS
Mendapatkan deskripsi untuk HybridConnection tertentu dalam ruang nama Relai.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmRelayHybridConnection** mendapatkan deskripsi untuk HybridConnection tertentu dalam ruang nama Relay.

## EXAMPLES

### Contoh 1
```
PS C:\>Get-AzureRmRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection

                              ybridConnections/TestHybridConnection
```
