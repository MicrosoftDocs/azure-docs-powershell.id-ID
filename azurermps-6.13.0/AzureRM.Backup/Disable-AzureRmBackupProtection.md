# Disable-AzureRmBackupProtection

## SYNOPSIS
Menonaktifkan proteksi untuk item terproteksi Cadangan.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Disable-AzureRmBackupProtection [-RemoveRecoveryPoints] [-Force] [-Item] <AzureRMBackupItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Disable-AzureRmBackupProtection** menonaktifkan proteksi untuk item yang diproteksi Azure Backup.
Cmdlet ini menghentikan pencadangan terjadwal reguler item.
Cmdlet ini dapat menghapus titik pemulihan yang sudah ada untuk item cadangan.
