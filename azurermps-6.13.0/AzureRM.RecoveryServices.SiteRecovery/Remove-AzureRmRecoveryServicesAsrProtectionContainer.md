# Remove-AzureRmRecoveryServicesAsrProtectionContainer

## SYNOPSIS
Menghapus Kontainer Proteksi yang ditentukan dari Fabric-nya.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmRecoveryServicesAsrProtectionContainer -InputObject <ASRProtectionContainer>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzureRmRecoveryServicesAsrProtectionContainer menghapus Kontainer Proteksi Site Recovery Azure yang ditentukan.

## EXAMPLES

### Contoh 1
```
PS C:\> Get-AzureRmRecoveryServicesAsrProtectionContainer -Name xxxxx  -Fabric $fabric
PS C:\> Remove-AzureRmRecoveryServicesAsrProtectionContainer -InputObject $protectionContainer
```

Memulai penghapusan wadah proteksi tertentu dan mengembalikan pekerjaan ASR yang digunakan untuk melacak operasi penghapusan.
