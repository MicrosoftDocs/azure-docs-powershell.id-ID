# New-AzureRmRecoveryServicesVault

## SYNOPSIS
Membuat vault Layanan Pemulihan yang baru.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzureRmRecoveryServicesVault** membuat vault Layanan Pemulihan baru.

## EXAMPLES

### Contoh 1
```
PS C:\> New-AzureRmRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Buat vault layanan pemulihan dalam grup sumber daya dan lokasi tertentu.
