# Get-AzureRmRecoveryServicesBackupProperty

## SYNOPSIS
Mendapatkan properti Cadangan.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmRecoveryServicesBackupProperty -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmRecoveryServicesBackupProperty** mendapatkan properti cadangan untuk vault Layanan Pemulihan.

## EXAMPLES

### Contoh 1
```
PS C:\> Get-AzureRmRecoveryServicesBackupProperty -Vault $vault
```

Dapatkan properti vault cadangan untuk vault.
