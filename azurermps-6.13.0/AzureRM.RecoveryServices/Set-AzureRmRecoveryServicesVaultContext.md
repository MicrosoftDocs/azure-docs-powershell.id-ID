# Set-AzureRmRecoveryServicesVaultContext

## SYNOPSIS
Mengatur konteks vault.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmRecoveryServicesVaultContext -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzureRmRecoveryServicesVaultContext** mengatur konteks vault untuk layanan Pemulihan Situs Azure.

## EXAMPLES

### Contoh 1
```
PS C:\> Set-AzureRmRecoveryServicesVaultContext -Vault $vault
```

Mengatur konteks vault.
