# Remove-AzureRmServiceFabricClusterCertificate

## SYNOPSIS
Hapus sertifikat kluster agar tidak digunakan untuk keamanan kluster.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmServiceFabricClusterCertificate -Thumbprint <String> [-Name] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Gunakan **Remove-AzureRmServiceFabricClusterCertificate** untuk menghapus sertifikat kluster dari kluster, selama ada sertifikat valid lain yang sudah digunakan dalam kluster.

## EXAMPLES

### Contoh 1
```
PS C:\> Remove-AzureRmServiceFabricClusterCertificate -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Thumbprint '5F3660C715EBBDA31DB1FFDCF508302348DE8E7A
```

Perintah ini menghapus sertifikat dengan thumbprint 5F3660C715EBBDA31DB1FFDCF508302348DE8E7A agar tidak digunakan untuk keamanan kluster.
