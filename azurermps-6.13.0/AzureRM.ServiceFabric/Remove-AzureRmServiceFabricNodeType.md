# Remove-AzureRmServiceFabricNodeType

## SYNOPSIS
Menghapus tipe node lengkap dari kluster.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmServiceFabricNodeType [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Gunakan **Remove-AzureRmServiceFabricNodeType** untuk menghapus semua node dari tipe node tertentu dan tipe node dari kluster. Perintah ini tidak dapat digunakan untuk menghapus tipe node utama.

## EXAMPLES

### Contoh 1
```
PS c:> Remove-AzureRmServiceFabricNodeType -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1'
```

Perintah ini akan menghapus NodeType 'nt1' dari kluster.
