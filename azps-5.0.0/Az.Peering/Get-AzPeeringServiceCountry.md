# Get-AzPeeringServiceCountry

## SYNOPSIS
Mencantumkan negara yang tersedia untuk layanan peering.

## SINTAKS


## DESKRIPSI
Mencantumkan negara yang tersedia untuk layanan peering.

