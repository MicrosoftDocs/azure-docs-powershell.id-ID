# Remove-AzMlCommitmentPlan

## SYNOPSIS
Menghapus rencana komitmen.

## SINTAKS

### RemoveByNameAndResourceGroup
```
Remove-AzMlCommitmentPlan -ResourceGroupName <String> -Name <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByObject
```
Remove-AzMlCommitmentPlan -MlCommitmentPlan <CommitmentPlan> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESKRIPSI
Menghapus Azure Machine Learning komitmen bisnis. Perhatikan bahwa paket komitmen yang memiliki asosiasi komitmen tidak bisa dihapus. Asosiasi komitmen hanya bisa dihapus oleh sumber daya target mereka. Misalnya, jika Anda menghapus Azure Machine Learning web perusahaan, komitmen yang mengaitkan layanan web dengan paket komitmen juga akan dihapus.

## CONTOH

### Contoh 1: Menghapus rencana komitmen
```
Remove-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName"
```
