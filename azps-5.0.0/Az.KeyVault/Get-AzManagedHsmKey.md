## DESKRIPSI
Cmdlet **Get-AzManagedHsmKey** mendapatkan tombol Azure Managed Hsm.
Cmdlet ini mendapatkan **Microsoft.Azure.Commands.KeyVault.Models.KeyBundle** tertentu atau daftar semua objek **KeyBundle** dalam Hsm terkelola atau menurut versi.

## CONTOH

### Contoh 1: Get all the keys in a managed HSM
```powershell
PS C:\> Get-AzManagedHsmKey -HsmName testmhsm
```

Nama Vault/HSM : nama testmhsm : testkey002 Versi : Id : Diaktifkan : False Kedaluwarsa https://testmhsm.managedhsm.azure.net:443/keys/testkey002 : 14/10/2022 8:13:29 AM Not Before : 14/10/2020 8:13:33 AM Created : 14/10/2020 8:14:01 AM Diperbarui : 14/10/2020 8:14:01 AM Tingkat Pemulihan : Recoverable+Purgeable Tags : Name Value Severity high Accounting true
