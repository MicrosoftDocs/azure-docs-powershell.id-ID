# Remove-AzFirewall

## SYNOPSIS
Hapus Firewall.

## SINTAKS

```
Remove-AzFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Remove-AzFirewall** menghapus Azure Firewall.

## CONTOH

### Contoh 1: Membuat dan menghapus Firewall
```powershell
New-AzFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Contoh ini membuat Firewall lalu menghapusnya. Untuk menyembunyikan perintah saat menghapus Firewall, gunakan bendera -Paksa.
