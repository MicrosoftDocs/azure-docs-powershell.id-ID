# Remove-AzP2sVpnGateway

## SYNOPSIS
Menghapus P2SVpnGateway yang sudah ada.

## SINTAKS

### ByP2SVpnGatewayName (Default)
```
Remove-AzP2sVpnGateway -ResourceGroupName <String> -Name <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByP2SVpnGatewayObject
```
Remove-AzP2sVpnGateway -InputObject <PSP2SVpnGateway> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByP2SVpnGatewayResourceId
```
Remove-AzP2sVpnGateway -ResourceId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Remove-AzP2sVpnGateway** memungkinkan Anda menghapus P2SVpnGateway yang sudah ada di bawah VirtualHub. Semua titik ke konektivitas klien situs akan gagal setelah P2SVpnGateway dihapus.

## CONTOH

### Contoh 1
```powershell
PS C:\> Remove-AzP2sVpnGateway -Name 683482ade8564515aed4b8448c9757ea-westus-gw-ResourceGroupName P2SCortexGATesting -Force -PassThru
```

Cmdlet **Remove-AzP2sVpnGateway** memungkinkan Anda menghapus P2SVpnGateway yang sudah ada di bawah VirtualHub.
