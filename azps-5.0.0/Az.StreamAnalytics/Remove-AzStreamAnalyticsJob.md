# Remove-AzStreamAnalyticsJob

## SYNOPSIS
Menghapus pekerjaan Analitik Stream.

## SINTAKS

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Remove-AzStreamAnalyticsJob** secara asinkron menghapus pekerjaan Analitik Streaming tertentu di Azure.

## CONTOH

### CONTOH 1: Menghapus pekerjaan
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Perintah ini menghapus pekerjaan StreamingJob.
