# Test-AzStreamAnalyticsInput

## SYNOPSIS
Menguji status koneksi input.

## SINTAKS

```
Test-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESKRIPSI
Cmdlet **Test-AzStreamAnalyticsInput** menguji kemampuan Stream Analytics untuk tersambung ke input.

## CONTOH

### Contoh 1: Menguji status koneksi aliran input
```powershell
PS C:\>Test-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Ini menguji status koneksi dari EntryStream input di StreamingJob.
