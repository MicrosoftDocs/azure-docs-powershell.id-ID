# Update-AzServiceFabricReliability

## SYNOPSIS
Perbarui tingkat keandalan tipe node utama dalam kluster.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.servicefabric/update-azservicefabricreliability) untuk informasi terkini.

## SYNTAX

```
Update-AzServiceFabricReliability [-ResourceGroupName] <String> [-Name] <String>
 -ReliabilityLevel <ReliabilityLevel> [-AutoAddNode] [-NodeType <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Gunakan **Update-AzServiceFabricReliability** untuk memperbarui keandalan tipe node utama dalam sebuah kluster.

## EXAMPLES

### Contoh 1
```powershell
PS c:> Update-AzServiceFabricReliability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -ReliabilityLevel Silver
```

Perintah ini mengubah tingkat keandalan tipe node utama menjadi perak.

### Contoh 2

Perbarui tingkat keandalan tipe node utama dalam kluster. (otomatisgenerated)

<!-- Aladdin Generated Example -->


```powershell
Update-AzServiceFabricReliability -AutoAddNode -Name 'Contoso01SFCluster' -ReliabilityLevel None -ResourceGroupName 'Group1'
```
