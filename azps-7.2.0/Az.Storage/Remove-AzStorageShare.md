# Remove-AzStorageShare

## SYNOPSIS
Menghapus berbagi file.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.storage/remove-azstorageshare) untuk informasi terkini.

## SYNTAX

### ShareName (Default)
```
Remove-AzStorageShare [-Name] <String> [-IncludeAllSnapshot] [-Force] [-PassThru] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Bagikan
```
Remove-AzStorageShare [-Share] <CloudFileShare> [-IncludeAllSnapshot] [-Force] [-PassThru]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzStorageShare** menghapus berbagi file.

## EXAMPLES

### Contoh 1: Hapus berbagi file
```
PS C:\>Remove-AzStorageShare -Name "ContosoShare06"
```

Perintah ini akan menghapus berbagi file bernama ContosoShare06.

### Contoh 2: Hapus berbagi file dan semua jepretan layarnya
```
PS C:\>Remove-AzStorageShare -Name "ContosoShare06" -IncludeAllSnapshot
```

Perintah ini menghapus berbagi file bernama ContosoShare06 dan semua snapshotnya.
