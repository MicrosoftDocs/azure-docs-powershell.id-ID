# Remove-AzIotHubKey

## SYNOPSIS
Menghapus Kunci IotHub.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan [lihat versi terbaru](/powershell/module/az.iothub/remove-aziothubkey) untuk informasi terkini.

## SYNTAX

### ResourceSet (Default)
```
Remove-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Remove-AzIotHubKey [-HubId] <String> [-KeyName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus Kunci IotHub.
Jika terdapat beberapa tombol dengan nama yang sama dengan yang pertama dalam daftar akan dihapus.

## EXAMPLES

### Contoh 1 Menghapus IotHub
```
PS C:\> Remove-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner1"
```

Menghapus kunci yang bernama iothubowner1 dari IotHub yang bernama "myiothub"
