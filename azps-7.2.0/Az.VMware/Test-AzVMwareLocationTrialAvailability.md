# Test-AzVMwareLocationTrialAvailability

## SYNOPSIS
Mengembalikan status uji coba untuk langganan menurut wilayah

## SYNTAX

```
Test-AzVMwareLocationTrialAvailability -Location <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Mengembalikan status uji coba untuk langganan menurut wilayah

## EXAMPLES

### Contoh 1: Periksa ketersediaan uji coba
```powershell
PS C:\> Test-AzVMwareLocationTrialAvailability -Location westcentralus

AvailableHost Status
------------- ------
0             TrialDisabled
```

Memeriksa ketersediaan uji coba
