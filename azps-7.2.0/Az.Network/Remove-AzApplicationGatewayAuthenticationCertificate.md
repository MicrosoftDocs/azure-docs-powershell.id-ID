# Remove-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Menghapus sertifikat autentikasi dari gateway aplikasi.

## SYNTAX

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzApplicationGatewayAuthenticationCertificate** menghapus sertifikat autentikasi dari gateway aplikasi Azure.

## EXAMPLES

### Contoh 1: Menghapus sertifikat autentikasi dari gateway aplikasi
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Perintah pertama mendapatkan gateway aplikasi bernama appGwName dan menyimpan hasilnya dalam $appgw variabel.
Perintah kedua menghapus sertifikat autentikasi bernama cert01 dari gateway aplikasi.
Perintah ketiga memperbarui gateway aplikasi.
