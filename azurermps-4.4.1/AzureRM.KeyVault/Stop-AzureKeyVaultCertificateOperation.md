# Stop-AzureKeyVaultCertificateOperation

## SYNOPSIS
Membatalkan operasi sertifikat di kunci vault.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Stop-AzureKeyVaultCertificateOperation** membatalkan operasi sertifikat di layanan Azure Key Vault.
