# Disable-AzureRmSchedulerJobCollection

## SYNOPSIS
Menonaktifkan kumpulan pekerjaan.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Disable-AzureRmSchedulerJobCollection -ResourceGroupName <String> -JobCollectionName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Disable-AzureRmSchedulerJobCollection** menonaktifkan kumpulan pekerjaan di Azure Scheduler.
