# Remove-AzureRmADUser

## SYNOPSIS
Menghapus pengguna direktori aktif.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmADUser -UPNOrObjectId <String> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus pengguna direktori aktif (akun kerja/sekolah juga dikenal sebagai org-id).
