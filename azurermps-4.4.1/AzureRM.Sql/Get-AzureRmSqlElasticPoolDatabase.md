# Get-AzureRmSqlElasticPoolDatabase

## SYNOPSIS
Mendapatkan database elastis dalam kolam elastis dan nilai propertinya.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlElasticPoolDatabase** mendapatkan database elastis dalam kolam elastis dan nilai propertinya.
Anda dapat menentukan nama database elastis di Azure SQL Database untuk melihat nilai properti hanya untuk database itu.

## EXAMPLES

### Contoh 1: Mendapatkan semua database dalam pool elastis
```
PS C:\>Get-AzureRmSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Perintah ini akan mendapatkan semua database dalam kolam elastis bernama ElastisPool01.
