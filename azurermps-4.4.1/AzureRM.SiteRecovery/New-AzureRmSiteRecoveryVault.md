# New-AzureRmSiteRecoveryVault

## SYNOPSIS
Membuat vault layanan Pemulihan Situs.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmSiteRecoveryVault -Name <String> -ResourceGroupName <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzureRmSiteRecoveryVault** membuat vault layanan Pemulihan Situs Azure.
