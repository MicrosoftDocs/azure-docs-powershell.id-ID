# Remove-AzureRmEnvironment

## SYNOPSIS
Menghapus titik akhir dan metadata untuk terhubung ke instans Azure tertentu.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmEnvironment [-Name] <String> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzureRmEnvironment menghapus informasi metadata dan titik akhir untuk tersambung ke contoh Azure tertentu.
