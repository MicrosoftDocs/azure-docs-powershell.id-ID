# Publish-AzureRmCdnEndpointContent

## SYNOPSIS
Memuat konten ke titik akhir.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### Kumpulan Parameter untuk parameter bidang (Default)
```
Publish-AzureRmCdnEndpointContent -EndpointName <String> -ProfileName <String> -ResourceGroupName <String>
 -LoadContent <String[]> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Kumpulan Parameter untuk parameter objek
```
Publish-AzureRmCdnEndpointContent -CdnEndpoint <PSEndpoint> -LoadContent <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Publish-AzureRmCdnEndpointContent** memuat konten dari server origin untuk titik akhir Azure Content Delivery Network (CDN).
