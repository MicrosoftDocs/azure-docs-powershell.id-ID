# Get-AzureRmCdnCustomDomain

## SYNOPSIS
Mendapatkan domain CDN kustom.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### Kumpulan Parameter untuk parameter bidang (Default)
```
Get-AzureRmCdnCustomDomain [-CustomDomainName <String>] -EndpointName <String> -ProfileName <String>
 -ResourceGroupName <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Kumpulan Parameter untuk parameter objek
```
Get-AzureRmCdnCustomDomain [-CustomDomainName <String>] -CdnEndpoint <PSEndpoint>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmCdnCustomDomain** mendapatkan domain kustom Azure Content Delivery Network (CDN) dan pengaturan terkait.
