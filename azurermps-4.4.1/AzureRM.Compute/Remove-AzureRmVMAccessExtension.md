# Remove-AzureRmVMAccessExtension

## SYNOPSIS
Menghapus ekstensi VMAccess dari komputer virtual.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmVMAccessExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmVMAccessExtension** menghapus Ekstensi Mesin Virtual Access (VMAccess) Virtual Machine Extension dari mesin virtual.
