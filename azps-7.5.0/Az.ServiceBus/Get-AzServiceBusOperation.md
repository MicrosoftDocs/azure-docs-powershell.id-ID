# Get-AzServiceBusOperation

## SYNOPSIS
Mencantumkan Operasi ServiceBus yang didukung

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.servicebus/get-azservicebusoperation) untuk informasi terbaru.

## SYNTAX

```
Get-AzServiceBusOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzServiceBusOperation** Mencantumkan Operasi yang didukung ServiceBus.

## EXAMPLES

### Contoh 1
```powershell
Get-AzServiceBusOperation
```

Mencantumkan operasi yang didukung ServiceBus
