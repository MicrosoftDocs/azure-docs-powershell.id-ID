# Remove-AzStackHCIRemoteSupport

## SYNOPSIS
Menghapus Dukungan Jarak Jauh.

## SYNTAX

```
Remove-AzStackHCIRemoteSupport [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Remove-AzStackHCIRemoteSupport menghapus instalan modul Penyebaran Dukungan Jarak Jauh.

## EXAMPLES

### CONTOH 1
```powershell
Remove-AzStackHCIRemoteSupport
```
