# New-AzDiskAccess

## SYNOPSIS
Membuat sumber daya Akses Disk

## SYNTAX

```
New-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzDiskAccess** membuat sumber daya Akses Disk

## EXAMPLES

### Contoh 1
```powershell
New-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" -Location "NorthCentralUS"
```

Perintah ini akan membuat Akses Disk dengan properti tertentu.
