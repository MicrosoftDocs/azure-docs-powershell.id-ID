# Update-AzAvailabilitySet

## SYNOPSIS
Memperbarui set ketersediaan.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.compute/update-azavailabilityset) untuk informasi terbaru.

## SYNTAX

```
Update-AzAvailabilitySet [-AvailabilitySet] <PSAvailabilitySet> [[-Sku] <String>]
 [-ProximityPlacementGroupId <String>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Update-AzAvailabilitySet** memperbarui set ketersediaan.

## EXAMPLES

### Contoh 1
```powershell
Get-AzAvailabilitySet -ResourceGroupName 'ResourceGroup01' -Name 'AvSet01' | Update-AzAvailabilitySet -Tag @{key1 = "val1"};
```

Perintah ini memperbarui tag set ketersediaan bernama 'AvSet01' di grup sumber daya bernama 'ResourceGroup01'.
