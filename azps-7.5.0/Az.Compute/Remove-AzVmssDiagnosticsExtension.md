# Remove-AzVmssDiagnosticsExtension

## SYNOPSIS
Menghapus ekstensi diagnostik dari VMSS.

## SYNTAX

```
Remove-AzVmssDiagnosticsExtension [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzVmssDiagnosticsExtension** menghapus ekstensi diagnostik dari Virtual Machine Scale Set (VMSS).

## EXAMPLES

### Contoh 1: Menghapus ekstensi diagnostik dari VMSS
```powershell
Remove-AzVmssDiagnosticsExtension -VirtualMachineScaleSet $VMSS -Name $extName
```

Perintah ini menghapus ekstensi diagnostik dari VMSS.
