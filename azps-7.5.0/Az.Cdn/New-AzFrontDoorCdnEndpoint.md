# New-AzFrontDoorCdnEndpoint

## SYNOPSIS
Membuat titik akhir.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.cdn/new-azfrontdoorcdnendpoint) untuk informasi terbaru.

## SYNTAX

```
New-AzFrontDoorCdnEndpoint -EndpointName <String> -ProfileName <String> -ResourceGroupName <String>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Membuat titik akhir.

## EXAMPLES

### Contoh 1
```powershell
New-AzFrontDoorCdnEndpoint -EndpointName $endpointName -ProfileName $profileName -ResourceGroupName $resourceGroupName
```

Membuat titik akhir.
