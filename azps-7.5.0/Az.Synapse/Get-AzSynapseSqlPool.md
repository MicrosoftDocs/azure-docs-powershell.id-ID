# Get-AzSynapseSqlPool

## SYNOPSIS
Mendapatkan kumpulan SQL Synapse Analytics.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.synapse/get-azsynapsesqlpool) untuk informasi terbaru.

## SYNTAX

### GetByNameParameterSet (Default)
```
Get-AzSynapseSqlPool [-ResourceGroupName <String>] -WorkspaceName <String> [-Name <String>] [-Version <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseSqlPool [-Name <String>] [-Version <Int32>] -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseSqlPool [-Version <Int32>] -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzSynapseSqlPool** mendapatkan informasi tentang kumpulan SQL Azure Synapse Analytics.

## EXAMPLES

### Contoh 1
```powershell
Get-AzSynapseSqlPool -WorkspaceName ContosoWorkspace
```

Perintah ini mendapatkan semua kumpulan SQL di bawah ruang kerja.

### Contoh 2
```powershell
Get-AzSynapseSqlPool -WorkspaceName ContosoWorkspace -Name ContosoSqlPool
```

Perintah ini mendapatkan kumpulan SQL di bawah ruang kerja ContosoWorkspace dengan nama ContosoSqlPool.

### Contoh: 3
```powershell
$ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
$ws | Get-AzSynapseSqlPool
```

Perintah ini mendapatkan semua kumpulan SQL di bawah ruang kerja melalui alur.

### Contoh 4
```powershell
Get-AzSynapseSqlPool -ResourceId "/subscriptions/21686af7-58ec-4f4d-9c68-f431f4db4edd/resourceGroups/ContosoResourceGroup/providers/Microsoft.Synapse/workspaces/ContosoWorkspace/sqlPools/ContosoSqlPool"
```

Perintah ini mendapatkan kumpulan SQL dengan ID sumber daya yang ditentukan.
