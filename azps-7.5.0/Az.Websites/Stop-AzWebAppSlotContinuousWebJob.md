# Stop-AzWebAppSlotContinuousWebJob

## SYNOPSIS
Hentikan pekerjaan web berkelanjutan untuk slot penyebaran.

## SYNTAX

### Hentikan (Default)
```
Stop-AzWebAppSlotContinuousWebJob -AppName <String> -Name <String> -ResourceGroupName <String>
 -SlotName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### StopViaIdentity
```
Stop-AzWebAppSlotContinuousWebJob -InputObject <IWebsitesIdentity> [-DefaultProfile <PSObject>] [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Hentikan pekerjaan web berkelanjutan untuk slot penyebaran.

## EXAMPLES

### Contoh 1: Menghentikan pekerjaan web berkelanjutan untuk slot penyebaran
```powershell
PS C:\> Stop-AzWebAppSlotContinuousWebJob -ResourceGroupName webjob-rg-test -AppName appService-test01 -SlotName slot01 -Name slotcontinuousjob-01
```

Perintah ini menghentikan pekerjaan web berkelanjutan untuk aplikasi.

### Contoh 2: Menghentikan pekerjaan web berkelanjutan untuk slot penyebaran menurut alur
```powershell
PS C:\> Get-AzWebAppSlotContinuousWebJob -ResourceGroupName webjob-rg-test -AppName appService-test01 -SlotName slot01 -Name slotcontinuousjob-01 | Stop-AzWebAppSlotContinuousWebJob
```

Perintah ini menghentikan pekerjaan web berkelanjutan untuk slot penyebaran berdasarkan alur.
