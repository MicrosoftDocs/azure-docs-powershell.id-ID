## SYNOPSIS
HANYA UNTUK PENGGUNAAN INTERNAL - Berikan Dukungan Runtime untuk cmdlet Yang Dihasilkan AutoRest

```
Register-AzModule [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
HANYA UNTUK PENGGUNAAN INTERNAL - Berikan Dukungan Runtime untuk cmdlet Yang Dihasilkan AutoRest

## EXAMPLES

### Contoh 1
