# New-AzNetworkWatcherConnectionMonitorEndpointScopeItemObject

## SYNOPSIS
Membuat item cakupan titik akhir monitor koneksi.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.network/new-aznetworkwatcherconnectionmonitorendpointscopeitemobject) untuk informasi terbaru.

## SYNTAX

```
New-AzNetworkWatcherConnectionMonitorEndpointScopeItemObject [-DefaultProfile <IAzureContextContainer>]
 -Address <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet New-AzNetworkWatcherConnectionMonitorEndpointScopeItemObject membuat item cakupan titik akhir.

## EXAMPLES

### Contoh 1
```powershell
New-AzNetworkWatcherConnectionMonitorEndpointScopeItemObject -Address "10.0.1.0/24"
```
