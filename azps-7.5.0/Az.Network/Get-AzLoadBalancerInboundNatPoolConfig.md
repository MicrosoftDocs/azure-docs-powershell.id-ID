# Get-AzLoadBalancerInboundNatPoolConfig

## SYNOPSIS
Mendapatkan satu atau beberapa konfigurasi kumpulan NAT masuk dari load balancer.

## SYNTAX

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzLoadBalancerInboundNatPoolConfig** mendapatkan satu atau beberapa konfigurasi kumpulan NAT masuk dari load balancer.

## EXAMPLES

### 1: Dapatkan
```powershell
$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
$slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
