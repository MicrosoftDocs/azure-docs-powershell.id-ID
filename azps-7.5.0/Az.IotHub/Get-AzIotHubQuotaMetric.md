# Get-AzIotHubQuotaMetric

## SYNOPSIS
Mendapatkan Metrik Kuota untuk IotHub.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.iothub/get-aziothubquotametric) untuk informasi terbaru.

## SYNTAX

```
Get-AzIotHubQuotaMetric [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Mendapatkan Metrik Kuota untuk IotHub.

## EXAMPLES

### Contoh 1 Dapatkan Metrik Kuota
```powershell
Get-AzIotHubQuotaMetric -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Mendapatkan informasi Metrik Kuota untuk IotHub bernama "myiothub"
