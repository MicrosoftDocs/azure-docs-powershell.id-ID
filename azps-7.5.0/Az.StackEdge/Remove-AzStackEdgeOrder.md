# Remove-AzStackEdgeOrder

## SYNOPSIS
Menghapus pesanan untuk perangkat.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.stackedge/remove-azstackedgeorder) untuk informasi terbaru.

## SYNTAX

### DeleteByNameParameterSet (Default)
```
Remove-AzStackEdgeOrder [-ResourceGroupName] <String> [-DeviceName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzStackEdgeOrder -ResourceId <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzStackEdgeOrder -InputObject <PSStackEdgeOrder> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzStackEdgeOrder** menghapus pesanan yang ada untuk perangkat Stack Edge.

## EXAMPLES

### Contoh 1
```powershell
Remove-AzStackEdgeOrder -ResourceGroupName resourceGroupName -DeviceName deviceName
```
