# Remove-AzKustoDataConnection

## SYNOPSIS
Menghapus koneksi data dengan nama yang diberikan.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.kusto/remove-azkustodataconnection) untuk informasi terbaru.

## SYNTAX

### Hapus (Default)
```
Remove-AzKustoDataConnection -ClusterName <String> -DatabaseName <String> -Name <String>
 -ResourceGroupName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzKustoDataConnection -InputObject <IKustoIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Menghapus koneksi data dengan nama yang diberikan.

## EXAMPLES

### Contoh 1: Menghapus koneksi data yang sudah ada berdasarkan nama
```powershell
Remove-AzKustoDataConnection -ResourceGroupName "testrg" -ClusterName "testnewkustocluster" -DatabaseName "mykustodatabase" -DataConnectionName "mykustodataconnection"
```

Perintah di atas menghapus koneksi data bernama "mykustodataconnection" dalam database "mykustodatabase" dari kluster yang ada "testnewkustocluster" yang ditemukan di grup sumber daya "testrg"
