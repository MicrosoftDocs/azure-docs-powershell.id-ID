# Remove-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Menghapus rencana pemulihan ASR yang ditentukan dari vault Layanan Pemulihan.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.recoveryservices/remove-azrecoveryservicesasrrecoveryplan) untuk informasi terbaru.

## SYNTAX

### ByObject (Default)
```
Remove-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Remove-AzRecoveryServicesAsrRecoveryPlan -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzRecoveryServicesAsrRecoveryPlan** menghapus rencana pemulihan yang ditentukan dari vault Layanan Pemulihan.

## EXAMPLES

### Contoh 1
```powershell
$currentJob = Remove-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Memulai penghapusan rencana pemulihan yang ditentukan dan mengembalikan pekerjaan ASR yang digunakan untuk melacak operasi.
