# Start-AzRecoveryServicesAsrCancelFailoverJob

## SYNOPSIS
Memulai tindakan failover pembatalan untuk objek Site Recovery.

## SYNTAX

### ByRPIObject (Default)
```
Start-AzRecoveryServicesAsrCancelFailoverJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByRPObject
```
Start-AzRecoveryServicesAsrCancelFailoverJob -RecoveryPlan <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Start-AzRecoveryServicesAsrCancelFailoverJob** memulai proses failover pembatalan untuk objek Azure Site Recovery.

## EXAMPLES

### Contoh 1
```powershell
$currentJob = Start-AzRecoveryServicesAsrCancelFailoverJob -RecoveryPlan $RP
```

Memulai failover pembatalan untuk rencana pemulihan yang ditentukan dan mengembalikan pekerjaan ASR yang digunakan untuk melacak operasi.

### Contoh 2

Memulai tindakan failover pembatalan untuk objek Site Recovery. (dibuat otomatis)

```powershell
<!-- Aladdin Generated Example --> 
Start-AzRecoveryServicesAsrCancelFailoverJob -ReplicationProtectedItem $ReplicationProtectedItem
```
