# Get-AzRecoveryServicesAsrNetwork

## SYNOPSIS
Mendapatkan informasi tentang jaringan yang dikelola oleh Site Recovery untuk vault saat ini.

## SYNTAX

### ByFabricObject (Default)
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByFriendlyName
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> -FriendlyName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzRecoveryServicesAsrNetwork** mendapatkan informasi tentang jaringan Azure Site Recovery untuk brankas Azure Site Recovery saat ini.

## EXAMPLES

### Contoh 1
```powershell
$Networks = Get-AzRecoveryServicesAsrNetwork -Fabric $Fabric
```

Mendapatkan semua jaringan yang diketahui dalam fabric yang ditentukan.
