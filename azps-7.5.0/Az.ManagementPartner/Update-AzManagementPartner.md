# Update-AzManagementPartner

## SYNOPSIS
Memperbarui ID Microsoft Partner Network (MPN) dari pengguna atau perwakilan layanan terautentikasi saat ini.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.managementpartner/update-azmanagementpartner) untuk informasi terbaru.

## SYNTAX

```
Update-AzManagementPartner [-PartnerId] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Memperbarui ID Microsoft Partner Network (MPN) dari pengguna atau perwakilan layanan terautentikasi saat ini.

## EXAMPLES

### Contoh 1
```powershell
Update-AzManagementPartner -PartnerId 4977985
```

```output
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Memperbarui mitra manajemen ke mitra baru
