# Add-AzMlWebServiceRegionalProperty

## SYNOPSIS
Membuat properti layanan web regional.

## SYNTAX

```
Add-AzMlWebServiceRegionalProperty -ResourceGroupName <String> -Name <String> -Region <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Membuat properti regional Azure Machine Learning untuk layanan web yang sudah ada.

## EXAMPLES

### Contoh 1: Menambahkan properti regional baru untuk US Barat Tengah

```powershell
Add-AzMlWebServiceRegionalProperty -ResourceGroupName "myresourcegroup" -Name "mywebservicename" -Region westcentralus
```

Contoh perintah ini membuat properti regional untuk layanan web di wilayah "BARAT Tengah AS".
