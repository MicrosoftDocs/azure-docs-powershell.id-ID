# Remove-AzMaintenanceConfiguration

## SYNOPSIS
Menghapus rekaman Konfigurasi

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.maintenance/remove-azmaintenanceconfiguration) untuk informasi terbaru.

## SYNTAX

```
Remove-AzMaintenanceConfiguration [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus rekaman Konfigurasi Pemeliharaan

## EXAMPLES

### Contoh 1
```powershell
Remove-AzMaintenanceConfiguration -ResourceGroupName smdtest -Name workervmscentralus
```

```output
Remove-AzMaintenanceConfiguration operation
This cmdlet will remove the specified resource.  Do you want to continue?
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"):
```

Menghapus rekaman Konfigurasi Pemeliharaan
