# Remove-AzDataLakeStoreAccount

## SYNOPSIS
Menghapus akun Data Lake Store secara permanen.

## SYNTAX

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzDataLakeStoreAccount** menghapus akun Data Lake Store secara permanen.

## EXAMPLES

### Contoh 1: Menghapus akun Data Lake Store
```powershell
Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Perintah ini menghapus akun bernama ContosoADL dari Data Lake Store.
