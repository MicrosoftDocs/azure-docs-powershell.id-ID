# Remove-AzDataLakeStoreItemAcl

## SYNOPSIS
Menghapus ACL file atau folder di Data Lake Store.

## SYNTAX

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzDataLakeStoreItemAcl** menghapus daftar kontrol akses (ACL) file atau folder di Data Lake Store.

## EXAMPLES

### Contoh 1: Menghapus ACL dari folder
```powershell
Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Perintah ini menghapus ACL untuk direktori akar untuk akun ContosoADL.
