# Remove-AzSpatialAnchorsAccount

## SYNOPSIS
Menghapus Akun Spatial Anchors

## SYNTAX

### DefaultParameterSet (Default)
```
Remove-AzSpatialAnchorsAccount -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzSpatialAnchorsAccount -ResourceId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PipelineParameterSet
```
Remove-AzSpatialAnchorsAccount -InputObject <PSSpatialAnchorsAccount> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Hapus Akun Spatial Anchors tertentu dari Langganan dan Grup Sumber Daya tertentu.

## EXAMPLES

### Contoh 1
```powershell
Remove-AzSpatialAnchorsAccount -ResourceGroup rg1 -Name example
```

Hapus "contoh" Akun Spatial Anchors dari Langganan saat ini dan Grup Sumber Daya "rg1".
