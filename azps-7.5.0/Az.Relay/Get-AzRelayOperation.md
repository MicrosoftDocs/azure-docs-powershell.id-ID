# Get-AzRelayOperation

## SYNOPSIS
Mencantumkan Operasi Relay yang didukung

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.relay/get-azrelayoperation) untuk informasi terbaru.

## SYNTAX

```
Get-AzRelayOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzRelayOperation** Mencantumkan Operasi yang didukung Relay.
