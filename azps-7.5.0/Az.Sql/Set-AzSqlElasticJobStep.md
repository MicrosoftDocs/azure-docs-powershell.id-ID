## DESCRIPTION
Cmdlet Set-AzSqlElasticJobStep memperbarui langkah pekerjaan

## EXAMPLES

### Contoh 1: Memperbarui grup target langkah pekerjaan untuk pekerjaan
```powershell
$jobStep = Get-AzSqlElasticJobStep -ResourceGroupName rg -ServerName elasticjobserver -AgentName agent -JobName job1 -StepName step1
$jobStep | Set-AzSqlElasticJobStep -TargetGroupName tg2
```

```output
JobName StepName StepId TargetGroupName CredentialName Output ExecutionOptions   CommandText
------- -------- ------ --------------- -------------- ------ ----------------   -----------
job1    step1    1      tg2             cred1                 (43200,10,1,120,2) SELECT 1
```

### Contoh 2: Memperbarui skrip T-SQL langkah pekerjaan untuk pekerjaan
```powershell
$jobStep = Get-AzSqlElasticJobStep -ResourceGroupName rg -ServerName elasticjobserver -AgentName agent -JobName job1 -StepName step1
$jobStep | Set-AzSqlElasticJobStep -CommandText "SELECT 2"
```

```output
JobName StepName StepId TargetGroupName CredentialName Output ExecutionOptions   CommandText
------- -------- ------ --------------- -------------- ------ ----------------   -----------
job1    step1    1      tg1             cred1                 (43200,10,1,120,2) SELECT 2
```

Memperbarui langkah pekerjaan dari pekerjaan

### Contoh 3
