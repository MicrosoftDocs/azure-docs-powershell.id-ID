# New-AzSqlServerCommunicationLink

## SYNOPSIS
Membuat tautan komunikasi untuk transaksi database elastis antara dua server SQL Database.

## SYNTAX

```
New-AzSqlServerCommunicationLink -LinkName <String> -PartnerServer <String> [-AsJob] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzSqlServerCommunicationLink** membuat tautan komunikasi untuk transaksi database elastis antara dua server logis di Azure SQL Database.
Transaksi database elastis dapat mencakup database di salah satu server yang dipasangkan.
Anda dapat membuat lebih dari satu tautan di server.
Oleh karena itu, transaksi database elastis dapat menjangkau sejumlah besar server.

## EXAMPLES

### Contoh 1: Membuat tautan komunikasi
```powershell
New-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01" -PartnerServer "ContosoServer02"
```

Perintah ini membuat tautan bernama Link01 antara ContosoServer17 dan ContosoServer02.
