# Get-AzSqlDatabaseActivity

## SYNOPSIS
Mendapatkan status operasi database.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.sql/get-azsqldatabaseactivity) untuk informasi terbaru.

## SYNTAX

```
Get-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzSqlDatabaseActivity** mendapatkan status operasi database di Azure SQL Database.

## EXAMPLES

### Contoh 1: Dapatkan status untuk semua instans SQL Database
```powershell
Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Perintah ini mengembalikan status operasi semua instans SQL Database dalam kumpulan elastis bernama ElasticPool01.

### Contoh 2: Dapatkan status untuk semua operasi SQL Database
```powershell
Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Perintah ini mengembalikan status semua operasi SQL Database dalam database.
