# New-AzSentinelIncidentOwner

## SYNOPSIS
Buat objek Pemilik Insiden untuk memperbarui pemilik insiden.

## SYNTAX

```
New-AzSentinelIncidentOwner -AssignedTo <String> -Email <String> -ObjectId <String> -UserPrincipalName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzSentinelIncidentOwner** membuat objek Pemilik Insiden dalam memori untuk memperbarui insiden.

## EXAMPLES

### Contoh 1
```powershell
$Incident = Get-AzSentinelIncident -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId"
$owner = New-AzSentinelIncidentOwner -AssignedTo "First Last" -Email "user@domain.com" -Objectid "userobjectId" -UserPrincipalName "user@domain.com"
$Incident.Owner = $owner
$Incident | Set-AzSentinelIncident
```

Contoh ini membuat **IncidentOwner** dan memperbarui Insiden kepada pemilik baru.
