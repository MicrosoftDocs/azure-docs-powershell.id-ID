# Get-AzsOffer

## SYNOPSIS
Dapatkan daftar penawaran publik untuk penyedia akar.

## SYNTAX

```
Get-AzsOffer [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Dapatkan daftar penawaran publik untuk penyedia akar.

## EXAMPLES

