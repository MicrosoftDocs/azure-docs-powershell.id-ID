# Remove-AzsComputeQuota

## SYNOPSIS
Hapus kuota Hitung yang sudah ada.

## SYNTAX

### Hapus (Default)
```
Remove-AzsComputeQuota -Name <String> [-Location <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzsComputeQuota -InputObject <IComputeAdminIdentity> [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Hapus kuota Hitung yang sudah ada.

## EXAMPLES

### Contoh 1: Hapus Kuota Perhitungan
```powershell
PS C:\> Remove-AzsComputeQuota -Name "AComputeQuota"
```

Panggilan yang berhasil untuk menghapus kuota perhitungan tidak akan mengembalikan output apa pun
