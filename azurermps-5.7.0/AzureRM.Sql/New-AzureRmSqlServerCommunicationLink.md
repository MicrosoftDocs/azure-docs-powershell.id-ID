# New-AzureRmSqlServerCommunicationLink

## SYNOPSIS
Membuat link komunikasi untuk transaksi database elastis antara dua SQL Database lain.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmSqlServerCommunicationLink -LinkName <String> -PartnerServer <String> [-AsJob]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzureRmSqlServerCommunicationLink** membuat tautan komunikasi untuk transaksi database elastis antara dua server logika di Azure SQL Database.
Transaksi database elastis bisa mencakup database di salah satu server yang dipasangkan.
Anda bisa membuat lebih dari satu link di server.
Oleh karena itu, transaksi database elastis dapat membentang di sejumlah server yang lebih besar.

## EXAMPLES

### Contoh 1: Membuat tautan komunikasi
```
PS C:\>New-AzureRmSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01" -PartnerServer "ContosoServer02"
```

Perintah ini membuat tautan bernama Link01 antara ContosoServer17 dan ContosoServer02.
