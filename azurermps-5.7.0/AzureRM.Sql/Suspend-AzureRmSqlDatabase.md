# Suspend-AzureRmSqlDatabase

## SYNOPSIS
Menangguhkan sebuah SQL database Gudang Data.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Suspend-AzureRmSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Suspend-AzureRmSqlDatabase** menangguhkan database Azure SQL Gudang Data.

## EXAMPLES

### Contoh 1: Menangguhkan database Gudang Data SQL Azure
```
PS C:\>Suspend-AzureRmSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Perintah ini menangguhkan database Gudang Data Azure SQL aktif.
