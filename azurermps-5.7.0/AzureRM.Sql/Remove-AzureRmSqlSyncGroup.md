# Remove-AzureRmSqlSyncGroup

## SYNOPSIS
Menghapus Azure SQL Database Sinkronisasi Otomatis.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSqlSyncGroup [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmSqlSyncGroup** menghapus Azure SQL Database Sinkronisasi Grup.

## EXAMPLES

### Contoh 1: Menghapus grup sinkronisasi
```
PS C:\>Remove-AzureRmSqlSyncGroup -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -Name "syncGroup01"
```

Perintah ini menghapus grup Azure SQL Database bernama syncGroup01.
