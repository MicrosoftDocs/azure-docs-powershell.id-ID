# Get-AzureRmSqlElasticPoolAdvisor

## SYNOPSIS
Dapatkan satu atau beberapa Penasihat untuk Azure SQL Elastis Pool.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzureRmSqlElasticPoolAdvisor** mendapatkan satu atau beberapa Azure SQL Elastis Pool Advisor untuk Pool Elastis Azure SQL Elastis.
