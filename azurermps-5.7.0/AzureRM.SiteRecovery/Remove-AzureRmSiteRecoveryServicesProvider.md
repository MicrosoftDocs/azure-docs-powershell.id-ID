# Remove-AzureRmSiteRecoveryServicesProvider

## SYNOPSIS
Menghapus Penyedia Layanan Pemulihan Situs Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSiteRecoveryServicesProvider -ServicesProvider <ASRRecoveryServicesProvider> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmSiteRecoveryServicesProvider** menghapus Penyedia Layanan Pemulihan Situs Azure dari penyimpanan.
