# Remove-AzureRmDataLakeStoreItem

## SYNOPSIS
Menghapus file atau folder di Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]> [-Recurse] [-Clean]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmDataLakeStoreItem** menghapus file atau folder di Data Lake Store.

## EXAMPLES

### Contoh 1: Menghapus beberapa item
```
PS C:\>Remove-AzureRmDataLakeStoreItem -AccountName "ContosoADL" -Paths "/File01.txt","/MyFiles/File.csv"
```

Perintah ini akan menghapus file File01.txt File.csv dari Data Lake Store.
