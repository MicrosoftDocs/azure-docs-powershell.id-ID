# Stop-AzureRmAutomationJob

## SYNOPSIS
Menghentikan pekerjaan Otomatisasi.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Stop-AzureRmAutomationJob** menghentikan pekerjaan Otomatisasi Azure.
Tentukan pekerjaan Otomatisasi yang sedang berjalan.

## EXAMPLES

### Contoh 1: Berhenti pekerjaan
```
PS C:\>Stop-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Perintah ini menghentikan pekerjaan dengan ID yang ditentukan.
