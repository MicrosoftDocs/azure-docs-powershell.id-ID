## DESCRIPTION
Cmdlet **Get-AzureRmResource** mendapatkan sumber daya Azure.

## EXAMPLES

### Contoh 1: Mendapatkan semua sumber daya tipe tertentu
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceType microsoft.web/sites
```

Perintah ini mendapatkan sumber daya dari tipe microsoft.web/situs di bawah ResourceGroup11.

### Contoh 2: Dapatkan sumber daya menurut nama
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceName ContosoWebsite
```

Perintah ini mendapatkan sumber daya bernama ContosoWebsite di bawah ResourceGroup11.

### Contoh 3: Memperlihatkan semua status akun penyimpanan dalam Grup Sumber Daya
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceType Microsoft.ClassicStorage/storageAccounts -ExpandProperties |
   Select * -Expand Properties |
   Sort Name |
   Format-Table Name,Status*
```
