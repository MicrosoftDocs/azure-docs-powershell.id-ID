# Remove-AzureRmRedisCacheDiagnostics

## SYNOPSIS
Menonaktifkan diagnostik di Singgahan Azure Redis.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmRedisCacheDiagnostics [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmRedisCacheDiagnostics** menonaktifkan diagnostik di Singgahan Azure Redis.

## EXAMPLES

### Contoh 1: Menonaktifkan diagnostik
```
PS C:\>Remove-AzureRmRedisCacheDiagnostics -ResourceGroupName "ContosoResourceGroup" -Name "PeakCache" -Force
```

Perintah ini menonaktifkan diagnostik pada Cache Azure Redis tertentu.

Tindakan ini akan menonaktifkan diagnostik untuk semua Singgahan Azure Redis di kawasan yang sama untuk langganan.
