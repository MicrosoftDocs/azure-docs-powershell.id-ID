# Get-AzReservationHistory

## SYNOPSIS
Dapatkan `Reservation` riwayat revisi.

## SYNTAX

### CommandLine (Default)
```
Get-AzReservationHistory -ReservationOrderId <Guid> -ReservationId <Guid>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### PipeObject
```
Get-AzReservationHistory -Reservation <PSReservation> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Daftar semua revisi untuk `Reservation`.

## EXAMPLES

### Contoh 1
```powershell
Get-AzReservationHistory -ReservationOrderId "1111aaaa-b1b2-c0c2-d0d2-00000fffff" -ReservationId "00000000-ffff-ffff-0000-00000fffff"
```

Dapatkan riwayat revisi reservasi tertentu
