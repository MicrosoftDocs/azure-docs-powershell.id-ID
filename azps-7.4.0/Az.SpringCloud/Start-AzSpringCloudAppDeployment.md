# Start-AzSpringCloudAppDeployment

## SYNOPSIS
Mulai penyebaran.

## SYNTAX

### Mulai (Default)
```
Start-AzSpringCloudAppDeployment -AppName <String> -Name <String> -ResourceGroupName <String>
 -ServiceName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### StartViaIdentity
```
Start-AzSpringCloudAppDeployment -InputObject <ISpringCloudIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Mulai penyebaran.

## EXAMPLES

### Contoh 1: Mulai Spring Cloud Service menurut nama.
```powershell
Start-AzSpringCloud -ResourceGroupName spring-cloud-rg -ServiceName spring-cloud-service -AppName gateway -DeploymentName default
```

Mulai Layanan Awan Musim Semi berdasarkan nama.

### Contoh 2: Start Spring Cloud Service from pipe.
```powershell
Get-AzSpringCloudAppDeployment -ResourceGroupName spring-cloud-rg -ServiceName spring-cloud-service -AppName gateway -DeploymentName default | Start-AzSpringCloud
```

Mulai Layanan Awan Musim Semi dari pipa.
