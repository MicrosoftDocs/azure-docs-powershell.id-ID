# New-AzNetworkWatcherConnectionMonitorOutputObject

## SYNOPSIS
Buat objek tujuan output monitor koneksi.

## SYNTAX

```
New-AzNetworkWatcherConnectionMonitorOutputObject [-OutputType <String>] -WorkspaceResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet New-AzNetworkWatcherConnectionMonitorOutputObject membuat objek tujuan output monitor koneksi.

## EXAMPLES

### Contoh 1
```powershell
New-AzNetworkWatcherConnectionMonitorOutputObject -OutputType "workspace" -WorkspaceResourceId MyWSResourceId
```

```output
Type              : "workspace"
WorkspaceSettings : {
                      "WorkspaceResourceId": "MyWSResourceId"
                    }
```
