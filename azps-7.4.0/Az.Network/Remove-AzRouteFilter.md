# Remove-AzRouteFilter

## SYNOPSIS
Menghapus filter rute.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.network/remove-azroutefilter) untuk informasi terbaru.

## SYNTAX

```
Remove-AzRouteFilter -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzRouteFilter** menghapus filter rute.

## EXAMPLES

### Contoh 1
```powershell
Remove-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
```

Perintah menghapus filter rute bernama RouteFilter01 di grup sumber daya bernama ResourceGroup01.
