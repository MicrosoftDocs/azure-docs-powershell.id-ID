# Start-AzKustoCluster

## SYNOPSIS
Memulai kluster Kusto.

## SYNTAX

### Mulai (Default)
```
Start-AzKustoCluster -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### StartViaIdentity
```
Start-AzKustoCluster -InputObject <IKustoIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Memulai kluster Kusto.

## EXAMPLES

### Contoh 1: Memulai klaster Kusto
```powershell
Start-AzKustoCluster -ResourceGroupName testrg -Name testnewkustocluster
```

Perintah di atas memulai klaster Kusto.
