# Remove-AzDataLakeAnalyticsDataSource

## SYNOPSIS
Menghapus sumber data dari akun Data Lake Analytics.

## SYNTAX

### RemoveDataLakeStorageAccount
```
Remove-AzDataLakeAnalyticsDataSource [-Account] <String> [-DataLakeStore] <String> [-Force] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### RemoveBlobStorageAccount
```
Remove-AzDataLakeAnalyticsDataSource [-Account] <String> [-Blob] <String> [-Force] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzDataLakeAnalyticsDataSource** menghapus sumber data dari akun azure Data Lake Analytics.

## EXAMPLES

### Contoh 1: Menghapus sumber data
```powershell
Remove-AzDataLakeAnalyticsDataSource -Account "ContosoAdlAccount" -Blob "AzureStorage01"
```

Perintah ini menghapus sumber data bernama AzureStorage01 dari akun bernama ContosoAdlAccount.
