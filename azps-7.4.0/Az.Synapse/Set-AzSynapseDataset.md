# Set-AzSynapseDataset

## SYNOPSIS
Membuat atau memperbarui himpunan data di ruang kerja.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.synapse/set-azsynapsedataset) untuk informasi terbaru.

## SYNTAX

### SetByName (Default)
```
Set-AzSynapseDataset -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataset -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzSynapseDataset** membuat himpunan data atau memperbarui himpunan data yang ada di ruang kerja.

## EXAMPLES

### Contoh 1
```powershell
Set-AzSynapseDataset -WorkspaceName ContosoWorkspace -Name ContosoDataset -DefinitionFile "C:\\samples\\Dataset.json"
```

Perintah ini membuat himpunan data bernama ContosoDataset di ruang kerja bernama ContosoWorkspace.
Perintah mendasarkan himpunan data pada informasi dalam file Dataset.json.
