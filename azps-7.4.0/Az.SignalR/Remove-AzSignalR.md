# Remove-AzSignalR

## SYNOPSIS
Menghapus layanan SignalR.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.signalr/remove-azsignalr) untuk informasi terbaru.

## SYNTAX

### ResourceGroupParameterSet (Default)
```
Remove-AzSignalR [-ResourceGroupName <String>] [-Name] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzSignalR -ResourceId <String> [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzSignalR -InputObject <PSSignalRResource> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus layanan SignalR.

## EXAMPLES

### Menghapus layanan SignalR
```powershell
Remove-AzSignalR -ResourceGroupName myResourceGroup -Name mysignalr1 -PassThru
```

```output
True
```

### Menghapus semua layanan SignalR dari pipa
```powershell
Get-AzSignalR -ResourceGroupName myResourceGroup | Remove-AzSignalR
```
