# Restart-AzSignalR

## SYNOPSIS
Mulai ulang layanan SignalR.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.signalr/restart-azsignalr) untuk informasi terbaru.

## SYNTAX

### ResourceGroupParameterSet (Default)
```
Restart-AzSignalR [-ResourceGroupName <String>] [-Name] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Restart-AzSignalR -ResourceId <String> [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Restart-AzSignalR -InputObject <PSSignalRResource> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Mulai ulang layanan SignalR.

## EXAMPLES

### Memulai ulang layanan SignalR tertentu
```powershell
Restart-AzSignalR -ResourceGroupName myResourceGroup -Name mysignalr1 -PassThru
```

```output
True
```

Grup sumber daya default dapat diatur oleh \`Set-AzDefault -ResourceGroupName myResourceGroup\`.
