# Move-AzDataLakeStoreItem

## SYNOPSIS
Memindahkan atau mengganti nama file atau folder di Penyimpanan Data Lake.

## SYNTAX

```
Move-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-Destination] <DataLakeStorePathInstance> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Move-AzDataLakeStoreItem** memindahkan atau mengganti nama file atau folder di Penyimpanan Data Lake.

## EXAMPLES

### Contoh 1: Memindahkan dan mengganti nama item
```powershell
Move-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/Original/Path/File.txt" -Destination "/New/Path/RenamedFile.txt"
```

Perintah ini mengganti nama item File.txt menjadi RenamedFile.txt dan memindahkannya ke folder lain.
