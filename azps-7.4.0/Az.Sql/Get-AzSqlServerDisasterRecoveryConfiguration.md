# Get-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Mendapatkan konfigurasi pemulihan sistem server database.

## SYNTAX

```
Get-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzSqlServerDisasterRecoveryConfiguration** mendapatkan konfigurasi pemulihan sistem server database SQL.
