# Get-AzSqlInstanceDatabaseLogReplay

## SYNOPSIS
Mendapatkan status layanan Pemutaran Ulang Log.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.sql/get-azsqlinstancedatabaselogreplay) untuk informasi terbaru.

## SYNTAX

```
Get-AzSqlInstanceDatabaseLogReplay [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzSqlInstanceDatabaseLogReplay** mendapatkan status layanan Pemutaran Ulang Log pada database tertentu.

## EXAMPLES

### Contoh 1
```powershell
Get-AzSqlInstanceDatabaseLogReplay -ResourceGroupName "ResourceGroup01" -InstanceName "ManagedInstance01" -Name "ManagedDatabaseName"
```

Perintah ini akan mendapatkan status layanan pemutaran ulang log pada database tertentu.
