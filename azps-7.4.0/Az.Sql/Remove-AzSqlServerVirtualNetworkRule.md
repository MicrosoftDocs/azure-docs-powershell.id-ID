# Remove-AzSqlServerVirtualNetworkRule

## SYNOPSIS
Menghapus Aturan Virtual Network Server Azure SQL.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.sql/remove-azsqlservervirtualnetworkrule) untuk informasi terbaru.

## SYNTAX

```
Remove-AzSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> [-AsJob] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Perintah ini menghapus Aturan Virtual Network Server Azure SQL.

## EXAMPLES

### Contoh 1
```powershell
$virtualNetworkRule = Remove-AzSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName
```

Menghapus aturan jaringan virtual Azure SQL Server yang ada
