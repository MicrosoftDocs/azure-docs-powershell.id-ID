# Remove-AzRecoveryServicesAsrReplicationProtectedItem

## SYNOPSIS
Menghentikan/Menonaktifkan replikasi untuk item yang diproteksi azure Site Recovery replikasi.

## SYNTAX

```
Remove-AzRecoveryServicesAsrReplicationProtectedItem -InputObject <ASRReplicationProtectedItem>
 [-WaitForCompletion] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzRecoveryServicesAsrReplicationProtectedItem** menonaktifkan replikasi item yang diproteksi Azure Site Recovery replikasi yang ditentukan.
Operasi ini menyebabkan replikasi berhenti untuk item yang diproteksi.

## EXAMPLES

### Contoh 1
```powershell
$currentJob = Remove-AzRecoveryServicesAsrReplicationProtectedItem -ReplicationProtectedItem $ReplicationProtectedItem
```

Memulai operasi nonaktifkan replikasi untuk item yang diproteksi replikasi tertentu dan mengembalikan pekerjaan ASR yang digunakan untuk melacak operasi.
