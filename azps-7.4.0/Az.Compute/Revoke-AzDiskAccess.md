# Revoke-AzDiskAccess

## SYNOPSIS
Mencabut akses ke diska.

## SYNTAX

```
Revoke-AzDiskAccess [-ResourceGroupName] <String> [-DiskName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Revoke-AzDiskAccess** mencabut akses ke disk.

## EXAMPLES

### Contoh 1
```powershell
Revoke-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'
```

Mencabut akses ke disk bernama 'Disk01' dalam grup sumber daya bernama 'ResourceGroup01'
