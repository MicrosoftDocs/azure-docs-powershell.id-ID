# Remove-AzVMNetworkInterface

## SYNOPSIS
Menghapus antarmuka jaringan dari mesin virtual.

## SYNTAX

```
Remove-AzVMNetworkInterface [-VM] <PSVirtualMachine> [[-NetworkInterfaceIDs] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzVMNetworkInterface** menghapus antarmuka jaringan dari mesin virtual.

## EXAMPLES

### Contoh 1
```powershell
Remove-AzVMNetworkInterface -VM $vmConfig
$vmConfig.NetworkProfile
```

```Output
NetworkInterfaces NetworkApiVersion NetworkInterfaceConfigurations
----------------- ----------------- ------------------------------
{}
```
