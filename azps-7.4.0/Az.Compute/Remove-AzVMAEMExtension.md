# Remove-AzVMAEMExtension

## SYNOPSIS
Menghapus ekstensi AEM dari komputer virtual.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.compute/remove-azvmaemextension) untuk informasi terbaru.

## SYNTAX

```
Remove-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [[-OSType] <String>] [-NoWait] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzVMAEMExtension** menghapus ekstensi Azure Enhanced Monitoring (AEM) dari komputer virtual.

## EXAMPLES

### Contoh 1: Menghapus ekstensi AEM
```powershell
Remove-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Perintah ini menghapus ekstensi AEM untuk komputer virtual bernama contoso-server.
