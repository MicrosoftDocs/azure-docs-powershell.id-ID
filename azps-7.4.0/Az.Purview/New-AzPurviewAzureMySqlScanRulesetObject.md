# New-AzPurviewAzureMySqlScanRulesetObject

## SYNOPSIS
Buat objek dalam memori untuk AzureMySqlScanRuleset.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.purview/new-azpurviewazuremysqlscanrulesetobject) untuk informasi terbaru.

## SYNTAX

```
New-AzPurviewAzureMySqlScanRulesetObject -Kind <DataSourceType> [-Description <String>]
 [-ExcludedSystemClassification <String[]>] [-IncludedCustomClassificationRuleName <String[]>]
 [-Type <ScanRulesetType>] [<CommonParameters>]
```

## DESCRIPTION
Buat objek dalam memori untuk AzureMySqlScanRuleset.

## EXAMPLES

### Contoh 1: Membuat objek scanruleset kustom Azure MySQL
