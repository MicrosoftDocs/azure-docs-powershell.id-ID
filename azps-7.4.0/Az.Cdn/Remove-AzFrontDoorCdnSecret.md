# Remove-AzFrontDoorCdnSecret

## SYNOPSIS
Menghapus rahasia.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.cdn/remove-azfrontdoorcdnsecret) untuk informasi terbaru.

## SYNTAX

### ByFieldsParameterSet (Default)
```
Remove-AzFrontDoorCdnSecret -ProfileName <String> -ResourceGroupName <String> -SecretName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Remove-AzFrontDoorCdnSecret -ResourceId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObjectParameterSet
```
Remove-AzFrontDoorCdnSecret -Secret <PSAfdSecret> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Menghapus rahasia.

## EXAMPLES

### Contoh 1
```powershell
Remove-AzFrontDoorCdnSecret -ResourceId $secretResourceId
```

Menghapus rahasia.
