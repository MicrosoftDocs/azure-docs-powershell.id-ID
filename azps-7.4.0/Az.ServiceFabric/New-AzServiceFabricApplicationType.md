# New-AzServiceFabricApplicationType

## SYNOPSIS
Buat jenis aplikasi service fabric baru di bawah grup sumber daya dan kluster yang ditentukan.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.servicefabric/new-azservicefabricapplicationtype) untuk informasi terbaru.

## SYNTAX

```
New-AzServiceFabricApplicationType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet membuat jenis aplikasi service fabric baru di bawah grup sumber daya dan kluster yang ditentukan.

## EXAMPLES

### Contoh 1
```powershell
$resourceGroupName = "testRG"
$clusterName = "testCluster"
$appTypeName = "testAppType"
$appType = New-AzServiceFabricApplicationType -ResourceGroupName $resourceGroupName -ClusterName $clusterName -Name $appTypeName -Verbose
```

Contoh ini akan membuat jenis aplikasi baru "testAppType" di bawah grup sumber daya dan kluster yang ditentukan.
