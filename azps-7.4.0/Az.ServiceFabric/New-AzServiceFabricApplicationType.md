# New-AzServiceFabricApplicationType

## SYNOPSIS
Buat jenis aplikasi kain layanan baru di bawah kelompok dan kluster sumber daya yang ditentukan.

## SYNTAX

```
New-AzServiceFabricApplicationType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet membuat jenis aplikasi kain layanan baru di bawah kelompok dan kluster sumber daya yang ditentukan.

## EXAMPLES

### Contoh 1
```powershell
$resourceGroupName = "testRG"
$clusterName = "testCluster"
$appTypeName = "testAppType"
$appType = New-AzServiceFabricApplicationType -ResourceGroupName $resourceGroupName -ClusterName $clusterName -Name $appTypeName -Verbose
```

Contoh ini akan membuat jenis aplikasi baru "testAppType" di bawah grup sumber daya dan kluster yang ditentukan.
