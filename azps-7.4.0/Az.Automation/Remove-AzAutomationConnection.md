# Remove-AzAutomationConnection

## SYNOPSIS
Menghapus koneksi Automation.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.automation/remove-azautomationconnection) untuk informasi terbaru.

## SYNTAX

```
Remove-AzAutomationConnection [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzAutomationConnection** menghapus koneksi dari Azure Automation.

## EXAMPLES

### Contoh 1: Menghapus koneksi
```
PS C:\>Remove-AzAutomationConnection -AutomationAccountName "Contoso17" -Name "ContosoConnection" -ResourceGroupName "ResourceGroup01"
```

Perintah ini menghapus sertifikat bernama ContosoConnection di akun Automation bernama Contoso17.
