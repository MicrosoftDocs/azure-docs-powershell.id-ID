# Remove-AzAutomationHybridWorkerGroup

## SYNOPSIS
Menghapus grup hybrid worker dari Automation.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.automation/remove-azautomationhybridworkergroup) untuk informasi terbaru.

## SYNTAX

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzAutomationHybridWorkerGroup menghapus grup hybrid worker dari Automation.

## EXAMPLES

### Contoh 1
Perintah ini menghapus pekerja hibrid berdasarkan nama.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName"
```
