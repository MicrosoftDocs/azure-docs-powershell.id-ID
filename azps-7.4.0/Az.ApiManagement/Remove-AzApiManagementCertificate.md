# Remove-AzApiManagementCertificate

## SYNOPSIS
Menghapus sertifikat API Management.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.apimanagement/remove-azapimanagementcertificate) untuk informasi terbaru.

## SYNTAX

```
Remove-AzApiManagementCertificate -Context <PsApiManagementContext> -CertificateId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzApiManagementCertificate** menghapus sertifikat Azure API Management.

## EXAMPLES

### Contoh 1: Menghapus sertifikat
```powershell
$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
Remove-AzApiManagementCertificate -Context $ApiMgmtContext -CertificateId "0123456789"
```

Perintah ini menghapus sertifikat API Management yang ditentukan.
