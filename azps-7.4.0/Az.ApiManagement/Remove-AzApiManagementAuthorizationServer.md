# Remove-AzApiManagementAuthorizationServer

## SYNOPSIS
Menghapus server otorisasi.

> [!NOTE]
>Ini adalah versi sebelumnya dari dokumentasi kami. Silakan lihat [versi terbaru](/powershell/module/az.apimanagement/remove-azapimanagementauthorizationserver) untuk informasi terbaru.

## SYNTAX

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzApiManagementAuthorizationServer** menghapus server otorisasi Azure API Management.

## EXAMPLES

### Contoh 1: Menghapus server otorisasi
```powershell
$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid"
```

Perintah ini menghapus Server Otorisasi API Management yang ditentukan.
