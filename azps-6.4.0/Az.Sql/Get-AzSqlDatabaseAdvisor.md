# Get-AzSqlDatabaseAdvisor

## SYNOPSIS
Dapatkan satu atau beberapa Penasihat untuk suatu Azure SQL Database.

## SYNTAX

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzSqlDatabaseAdvisor** mendapatkan satu atau beberapa Azure SQL Database penasihat untuk mendapatkan Azure SQL Database.
