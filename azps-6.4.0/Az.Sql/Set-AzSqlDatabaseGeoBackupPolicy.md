# Set-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Mengatur kebijakan database geo backup.

## SYNTAX

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzSqlDatabaseGeoBackupPolicy** mengatur kebijakan geo backup yang didaftarkan ke database.
Ini adalah sumber daya Cadangan Azure yang digunakan untuk menentukan kebijakan penyimpanan cadangan.
