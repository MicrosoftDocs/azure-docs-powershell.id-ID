# Set-AzSqlDatabaseTransparentDataEncryption

## SYNOPSIS
Mengubah properti TDE untuk database.

## SYNTAX

```
Set-AzSqlDatabaseTransparentDataEncryption [-State] <TransparentDataEncryptionStateType> [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzSqlDatabaseTransparentDataEncryption** memodifikasi properti Transparent Data Encryption (TDE) dari database Azure SQL.
Untuk informasi selengkapnya, lihat Enkripsi Data Transparan dengan Azure SQL Database https://msdn.microsoft.com/library/dn948096 ( di Pustaka Jaringan Pengembang https://msdn.microsoft.com/library/dn948096) Microsoft.
Cmdlet ini juga didukung oleh layanan SQL Server Stretch Database di Azure.

## EXAMPLES

### Contoh 1: Mengaktifkan TDE untuk database
```
PS C:\>Set-AzSqlDatabaseTransparentDataEncryption -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -State Enabled
ResourceGroupName             ServerName                    DatabaseName                                          State
-----------------             ----------                    ------------                                          -----
ResourceGroup01               Server01                      Database01                                            Enabled
```

Perintah ini mengaktifkan TDE untuk database yang bernama Database01 di server yang bernama Server01.
