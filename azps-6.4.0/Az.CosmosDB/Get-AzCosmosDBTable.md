# Get-AzCosmosDBTable

## SYNOPSIS
Mendapatkan Tabel CosmosDB.

## SYNTAX

### ByNameParameterSet (Default)
```
Get-AzCosmosDBTable -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBTable [-Name <String>] -ParentObject <PSDatabaseAccountGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzCosmosDBTable** mendapatkan Tabel yang sudah ada.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Get-AzCosmosDBTable -AccountName {account} -Name {tableName} -ResourceGroupName {rgName}

Name    Id    Resource
{name}  {id}  Microsoft.Azure.Commands.CosmosDB.Models.PSTableGetPropertiesResource
```

Objek sumber daya berisi _rid, _ts, _ etag properti Tabel.
