# Get-AzSnapshot

## SYNOPSIS
Mendapatkan properti snapshot

## SYNTAX

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzSnapshot** mendapatkan properti snapshot.
