# Add-AzDataLakeStoreFirewallRule

## SYNOPSIS
Menambahkan aturan firewall ke akun Data Lake Store tertentu.

## SYNTAX

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Add-AzDataLakeStoreFirewallRule** menambahkan aturan firewall ke akun Data Lake Store tertentu.

## EXAMPLES

### Contoh 1: Menambahkan aturan firewall baru ke akun Data Lake Store
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Ini membuat aturan firewall baru yang disebut "MyRule" dalam akun "ContosoADL" dengan rentang IP 127.0.0.1 - 127.0.0.2
