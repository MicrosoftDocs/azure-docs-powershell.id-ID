# Get-AzP2sVpnGateway

## SYNOPSIS
Mendapatkan P2SVpnGateway yang sudah ada di bawah VirtualHub.

## SYNTAX

### ListBySubscriptionId (Default)
```
Get-AzP2sVpnGateway [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ListByResourceGroupName
```
Get-AzP2sVpnGateway [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Get-AzP2sVpnGateway** memungkinkan Anda mendapatkan P2SVpnGateway yang sudah ada di bawah VirtualHub.
