# Remove-AzureRmNetworkWatcher

## SYNOPSIS
Menghapus Pengawas Jaringan.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmNetworkWatcher -Name <String> -ResourceGroupName <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzureRmNetworkWatcher menghapus sumber daya Pengawas Jaringan.

## EXAMPLES

### -------------------------- Contoh 1: Membuat dan menghapus pengaturan Pengawas --------------------------
```
New-AzureRmResourceGroup -Name NetworkWatcherRG -Location westcentralus
New-AzureRmNetworkWatcher -Name NetworkWatcher_westcentralus -ResourceGroup NetworkWatcherRG -Location westcentralus
Remove-AzureRmNetworkWatcher -Name NetworkWatcher_westcentralus -ResourceGroup NetworkWatcherRG
```

Contoh ini membuat Pengawas Jaringan dalam grup sumber daya lalu segera menghapusnya. Perhatikan bahwa hanya satu Pengawas Jaringan yang dapat dibuat per kawasan per langganan.
Untuk menyembunyikan perintah saat menghapus jaringan virtual, gunakan bendera -Paksa.
