# Remove-AzureRmRouteTable

## SYNOPSIS
Menghapus tabel rute.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmRouteTable -ResourceGroupName <String> -Name <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmRouteTable** menghapus tabel rute Azure.

## EXAMPLES

### Contoh 1: Menghapus tabel rute
```
PS C:\>Remove-AzureRmRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Perintah ini menghapus tabel rute bernama RouteTable01 dalam grup sumber daya bernama ResourceGroup11.
Cmdlet meminta anda untuk konfirmasi sebelum menghapus tabel.
