# Set-AzureRmRouteTable

## SYNOPSIS
Menetapkan status tujuan untuk tabel rute.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzureRmRouteTable** menetapkan status tujuan untuk tabel rute Azure.
