# Remove-AzureRmLoadBalancer

## SYNOPSIS
Menghapus penyeimbang muat.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmLoadBalancer -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmLoadBalancer** menghapus penyeimbang muat Azure.

## EXAMPLES

### Contoh 1: Menghapus penyeimbang muat
```
PS C:\>Remove-AzureRmLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
```

Perintah ini menghapus penyeimbang muat yang bernama MyLoadBalancer dalam grup sumber daya bernama MyResourceGroup.
