# Remove-AzureRmWebApp

## SYNOPSIS
Menghapus Azure Web App.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### S1
```
Remove-AzureRmWebApp [-Force] [-ResourceGroupName] <String> [-Name] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzureRmWebApp [-Force] [-WebApp] <Site> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Hapus-AzureRmWebApp** menghapus Azure Web App yang disediakan grup sumber daya dan nama Web App.
Cmdlet ini, secara default, juga menghapus semua slot dan metrik.

## EXAMPLES

### Contoh 1: Menghapus Aplikasi Web
```
PS C:\>Remove-AzureRmWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Perintah ini menghapus Azure Web App bernama ContosoSite yang termasuk dalam grup sumber daya bernama Default-Web-WestUS.
