# Remove-AzureStorageContainerStoredAccessPolicy

## SYNOPSIS
Menghapus kebijakan akses tersimpan dari tempat penyimpanan Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureStorageContainerStoredAccessPolicy [-Container] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureStorageContainerStoredAccessPolicy** menghapus kebijakan akses tersimpan dari wadah penyimpanan Azure.

## EXAMPLES

### Contoh 1: Menghapus kebijakan akses yang disimpan dari wadah penyimpanan
```
PS C:\>Remove-AzureStorageContainerStoredAccessPolicy -Container "MyContainer" -Policy "Policy03"
```

Perintah ini menghapus kebijakan akses yang bernama Policy03 dari wadah tersimpan yang bernama MyContainer.
