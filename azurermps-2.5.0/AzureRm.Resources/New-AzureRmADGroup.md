# New-AzureRmADGroup

## SYNOPSIS
Membuat grup direktori aktif baru.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Membuat grup direktori aktif baru.

## EXAMPLES

### Contoh 1 - Membuat grup AD baru

```
PS C:\> New-AzureRmADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Membuat grup AD baru dengan nama "MyGroupDisplayName" dan nama panggilan email "myemail@domain.com".
