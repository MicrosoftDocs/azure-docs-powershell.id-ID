# Remove-AzureRmVMAccessExtension

## SYNOPSIS
Menghapus ekstensi VMAccess dari mesin virtual.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmVMAccessExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzureRmVMAccessExtension** menghapus Ekstensi Mesin Virtual Akses Mesin Virtual (VMAccess) dari mesin virtual.
