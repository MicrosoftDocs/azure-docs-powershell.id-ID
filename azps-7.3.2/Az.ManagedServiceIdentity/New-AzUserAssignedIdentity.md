# New-AzUserAssignedIdentity

## SYNOPSIS
Membuat Identitas yang Ditetapkan Pengguna baru atau memperbarui Identitas yang Ditetapkan Pengguna yang sudah ada.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.managedserviceidentity/new-azuserassignedidentity) untuk informasi terbaru.

## SYNTAX

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **New-AzUserAssignedIdentity** membuat Identitas yang Ditetapkan Pengguna baru. Ketika digunakan dengan identitas yang sudah ada, identitas diperbarui.
Untuk menambahkan tag Azure Resource Manager ke identitas, silakan gunakan cmdlet Set-AzResource.
