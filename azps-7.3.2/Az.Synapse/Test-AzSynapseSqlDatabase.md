# Test-AzSynapseSqlDatabase

## SYNOPSIS
Fitur ini berada dalam pratinjau terbatas, awalnya hanya dapat diakses untuk langganan tertentu. Memeriksa keberadaan database SQL Analitik Synapse.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.synapse/test-azsynapsesqldatabase) untuk informasi terbaru.

## SYNTAX

### TestByNameParameterSet (Default)
```
Test-AzSynapseSqlDatabase [-ResourceGroupName <String>] -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByParentObjectParameterSet
```
Test-AzSynapseSqlDatabase -Name <String> -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Test-AzSynapseSqlDatabase** memeriksa keberadaan database SQL Analitik Synapse.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> Test-AzSynapseSqlDatabase -WorkspaceName ContosoWorkspace -Name ContosoSqlDatabase
```

Perintah ini memeriksa keberadaan database SQL yang ditentukan.
