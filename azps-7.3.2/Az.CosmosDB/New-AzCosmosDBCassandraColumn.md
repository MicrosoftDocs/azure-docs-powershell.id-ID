# New-AzCosmosDBCassandraColumn

## SYNOPSIS
Membuat Kolom CosmosDB Cassandra baru.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.cosmosdb/new-azcosmosdbcassandracolumn) untuk informasi terbaru.

## SYNTAX

```
New-AzCosmosDBCassandraColumn -Name <String> -Type <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
**New-AzCosmosDBCassandraColumn** membuat Kolom CosmosDB Cassandra baru.

## EXAMPLES

### Contoh 1
```powershell
PS C:\> New-AzCosmosDBCassandraColumn -Name "name" -Type int

Name Type
---- ----
name int
```
