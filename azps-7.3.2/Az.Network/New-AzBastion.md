## DESCRIPTION
Membuat sumber daya sdm. Ini akan membutuhkan Alamat Ip Publik dan VirtualNetwork. Harus ada subnet dengan nama AzureBastionSubnet di VirtualNetwork ini.Alamat Ip Pubic harus dibuat dengan Sku Standard.
