# Set-AzLocalNetworkGateway

## SYNOPSIS
Mengubah gateway jaringan lokal.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.network/set-azlocalnetworkgateway) untuk informasi terbaru.

## SYNTAX

```
Set-AzLocalNetworkGateway -LocalNetworkGateway <PSLocalNetworkGateway> [-AddressPrefix <String[]>]
 [-Asn <UInt32>] [-BgpPeeringAddress <String>] [-PeerWeight <Int32>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzLocalNetworkGateway** mengubah gateway jaringan lokal.

## EXAMPLES

### Contoh 1
Mengatur konfigurasi untuk gateway yang sudah ada


```
$lgw = Get-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```
