# Remove-AzRelayHybridConnection

## SYNOPSIS
Menghapus HybridConnection dari ruang nama HybridConnection yang ditentukan.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.relay/remove-azrelayhybridconnection) untuk informasi terbaru.

## SYNTAX

```
Remove-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzRelayHybridConnection** menghapus HybridConnection dari ruang nama Relay yang ditentukan.

## EXAMPLES

### Contoh 1
```
PS C:\> Remove-AzRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Menghapus HybridConnection `TestHybridConnection` dari ruang `TestNameSpace-Relay1`nama .
