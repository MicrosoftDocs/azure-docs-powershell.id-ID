# Set-AzVMBackupExtension

## SYNOPSIS
Mengatur properti ekstensi cadangan pada mesin maya.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.compute/set-azvmbackupextension) untuk informasi terbaru.

## SYNTAX

```
Set-AzVMBackupExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Tag] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```
