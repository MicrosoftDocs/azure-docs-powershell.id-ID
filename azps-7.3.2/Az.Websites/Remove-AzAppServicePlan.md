# Remove-AzAppServicePlan

## SYNOPSIS
Menghapus paket Azure App Service.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.websites/remove-azappserviceplan) untuk informasi terbaru.

## SYNTAX

### S1
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-AppServicePlan] <PSAppServicePlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Remove-AzAppServicePlan** menghapus rencana Azure App Service.

## EXAMPLES

### Contoh 1: Menghapus paket App Service
```
PS C:\>Remove-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Perintah ini menghapus paket Azure App Service bernama ContosoASP yang termasuk dalam grup sumber daya bernama Default-Web-WestUS.
