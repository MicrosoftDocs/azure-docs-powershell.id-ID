# Set-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Mengubah konfigurasi pemulihan server database.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.sql/set-azsqlserverdisasterrecoveryconfiguration) untuk informasi terbaru.

## SYNTAX

```
Set-AzSqlServerDisasterRecoveryConfiguration -VirtualEndpointName <String> [-Failover] [-AllowDataLoss]
 [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzSqlServerDisasterRecoveryConfiguration memodifikasi** konfigurasi pemulihan server database SQL.
