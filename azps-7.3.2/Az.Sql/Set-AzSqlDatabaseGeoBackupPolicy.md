# Set-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Mengatur kebijakan pencadangan geo database.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.sql/set-azsqldatabasegeobackuppolicy) untuk informasi terbaru.

## SYNTAX

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Cmdlet **Set-AzSqlDatabaseGeoBackupPolicy** mengatur kebijakan cadangan geo yang didaftarkan ke database.
Ini adalah sumber daya Azure Backup yang digunakan untuk menentukan kebijakan penyimpanan cadangan.

## EXAMPLES

### Contoh 1

Mengatur kebijakan pencadangan geo database. (autogenerasi)

<!-- Aladdin Generated Example -->
```powershell
Set-AzSqlDatabaseGeoBackupPolicy -DatabaseName db1 -ResourceGroupName myresourcegroup -ServerName s1 -State Disabled
```
