# Remove-AzSqlServerKeyVaultKey

## SYNOPSIS
Menghapus kunci Key Vault dari server SQL.

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.sql/remove-azsqlserverkeyvaultkey) untuk informasi terbaru.

## SYNTAX

```
Remove-AzSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cmdlet Remove-AzSqlServerKeyVaultKey menghapus kunci Key Vault dari server SQL yang ditentukan.
Perhatikan bahwa izin server SQL ke kubah kunci tidak diubah.
Untuk mengubah izin, gunakan Set-AzKeyVaultAccessPolicy.
Perhatikan bahwa cmdlet ini tidak membuat perubahan pada Key Vault.
Untuk menghapus kunci dari Key Vault, gunakan Remove-AzKeyVaultKey.

## EXAMPLES

### Contoh 1: Menghapus kunci Key Vault
```
PS C:\> Remove-AzSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Perintah ini menghapus kunci Key Vault dengan Id 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' dari server yang ditentukan.
