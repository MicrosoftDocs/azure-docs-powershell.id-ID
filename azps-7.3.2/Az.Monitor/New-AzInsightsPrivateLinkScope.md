# New-AzInsightsPrivateLinkScope

## SYNOPSIS
buat lingkup tautan privat

> [!NOTE]
>Ini adalah versi dokumentasi kami sebelumnya. Silakan lihat [versi terbaru](/powershell/module/az.monitor/new-azinsightsprivatelinkscope) untuk informasi terbaru.

## SYNTAX

```
New-AzInsightsPrivateLinkScope -Location <String> -ResourceGroupName <String> -Name <String> [-Tags <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
buat lingkup tautan privat

## EXAMPLES

### Contoh 1
```powershell
New-AzInsightsPrivateLinkScope -Location "eastus" -ResourceGroupName "rg_name" -Name "scope_name"
```

buat lingkup tautan privat dengan nama "scope_name" di bawah grup sumber daya "rg_name" di lokasi "eastus"
