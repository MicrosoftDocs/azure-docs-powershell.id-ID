# Get-AzLegacyPeering

## SYNOPSIS
Digunakan untuk Mengonversi sumber daya Peering Warisan ke Sumber Daya Azure Resource Management (ARM). 

## SYNTAX

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
Perintah ini digunakan untuk menampilkan sumber daya Peering warisan yang semuanya Anda konversi menjadi Sumber Daya Azure Resource Management (ARM).
