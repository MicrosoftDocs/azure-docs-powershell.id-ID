# New-AzsDiskMigrationJob

## SYNOPSIS
Memulai pekerjaan migrasi disk terkelola untuk melakukan migrasi disk terkelola ke berbagi tujuan yang ditentukan.

## SYNTAX

```
New-AzsDiskMigrationJob [-Disks] <Disk[]> [-TargetShare] <String> [[-Location] <String>] [-Name] <String>
 [<CommonParameters>]
```

## DESCRIPTION
Membuat pekerjaan migrasi disk.

## EXAMPLES

### -------------------------- CONTOH 1 --------------------------
```
New-AzsDiskMigrationJob -Name "MyMigrationJob" -Disks $disks -location local -TargetShare "\\SU1FileServer.azurestack.local\SU1_ObjStore"
```

Memulai pekerjaan migrasi disk terkelola untuk 20 disk pertama
